# ACCEPTANCE — 验收标准

## 功能验收

### AC-1: 书名输入
- [ ] 创建项目时有书名输入框（可选）
- [ ] 留空时显示"AI将为您起名"
- [ ] 用户可以手动输入书名（1-30字符）

### AC-2: AI 生成书名
- [ ] 大纲完成后自动生成5个书名候选
- [ ] 每个书名包含吸引力分析
- [ ] 用户可以选择一个或要求重新生成

### AC-3: 简介生成
- [ ] 自动生成3个版本的简介（50字/150字/300字）
- [ ] 简介包含核心冲突和看点
- [ ] 用户可以编辑简介内容

### AC-4: 标签推荐
- [ ] 自动推荐5个标签
- [ ] 标签与题材/平台匹配
- [ ] 用户可以增删标签

### AC-5: 分类匹配
- [ ] 自动匹配番茄分类
- [ ] 用户可以修改分类

### AC-6: API 接口
- [ ] GET /api/projects/{id}/metadata
- [ ] POST /api/projects/{id}/metadata
- [ ] POST /api/projects/{id}/metadata/regenerate

### AC-7: Pipeline 集成
- [ ] 大纲后自动进入元数据生成
- [ ] 元数据生成后才进入正文

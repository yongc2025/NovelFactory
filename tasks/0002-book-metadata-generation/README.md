# Task 0002: 书籍元数据生成

## 概述

为生成的小说自动创建完整的书籍元数据：书名、简介、标签、分类。

## 问题

当前系统只生成故事内容（世界观/角色/大纲/正文），但发布到番茄/小红书需要：
- 书名（1-30字符，要有吸引力）
- 简介（50-500字符，决定点击率）
- 标签（3-5个，影响推荐）
- 分类（番茄男频/女频 → 一级 → 二级）

用户没有地方输入书名，AI也没有帮起名。

## 目标

1. 创建项目时必须有书名字段（用户填写 或 AI生成）
2. 大纲完成后自动生成简介（50-500字符）
3. 自动推荐标签（3-5个）
4. 自动匹配番茄/小红书分类

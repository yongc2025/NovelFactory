# 01 世界总定位

## 一句话世界观

这是一个进入末法时代的魔法工业世界。高魔时代曾经支撑起辉煌的魔导工业，但随着魔力浓度下降、魔石杂质增加、高阶法师断代和旧设备老化，传统依赖高魔环境、高品质材料、大师经验和行会认证的旧工业体系正在失效。

## 世界核心矛盾

```text
旧魔导工业仍垄断认证、秘方和市场，
但旧体系赖以运行的高魔环境已经不存在。
```

这不是“魔法没落后大家都变蠢”，而是：

```text
旧体系曾经有效，
但它没有建立足够的测量、记录、标准、材料数据库、低成本替代和批量培训机制，
所以在末法时代暴露出结构性脆弱。
```

## 主角切入点

陆衡不能靠现代知识直接碾压世界。他能切入，是因为他擅长把问题从“神秘天赋”拆成：

```text
材料问题
结构问题
工艺问题
热效率问题
稳定性问题
批次问题
记录问题
岗位分工问题
```

他的优势不是知道答案，而是建立验证路径。

# 03 时代背景：高魔时代到末法时代

## 高魔时代

高魔时代曾经拥有成熟而辉煌的魔导工业。

典型成果包括：

- 魔晶炉
- 魔导水泵
- 贵族供暖系统
- 浮空运输
- 自动铭文机
- 行会认证工坊
- 贵族家族秘方
- 高阶法师维护网络

高魔时代的成功依赖：

- 高浓度环境魔力
- 高品质魔石
- 高阶法师数量充足
- 大师经验传承
- 行会认证体系
- 贵族垄断市场
- 家族秘方和工坊传统

## 末法时代

末法时代不是一夜降临，而是长期衰退后进入显著恶化阶段。

表现包括：

- 环境魔力浓度下降。
- 魔石杂质比例上升。
- 高阶法师断代。
- 旧魔导设备维护成本上升。
- 魔导器故障率提高。
- 平民重新被挤出魔法工业消费体系。
- 贵族和法师行会仍用旧时代标准控制认证和市场。

## 第一卷禁止提前揭示

第一卷不揭示末法历史真相。

第一卷只允许给出表层现象：

```text
炉子坏了。
魔石差了。
维修贵了。
行会认证慢了。
领民用不起旧设备了。
灰石领快撑不过寒潮了。
```

历史原因、过度开采、旧魔法工业根源性危机等内容，留到后续卷逐步展开。

# 04 灰石领

## 基本定位

灰石领是莱恩·格雷名下的边境破落领地。

它不是战略核心区，而是旧贵族体系边缘地带。

第一卷中，灰石领的功能是：

```text
一个足够贫穷、足够寒冷、足够被旧体系遗忘的小社会，
让主角必须从最低成本、最现实的生存问题开始改造世界。
```

## 地理与气候

- 位于边境寒冷地带。
- 冬季长，寒潮危险。
- 交通不便，物资调入成本高。
- 周边有低品位矿石、粘土、灰石、木材等基础资源。
- 农业产出有限，领民抗风险能力差。

## 资源条件

可用资源：

- 灰石：可用于建筑、炉体填料、粗加工材料。
- 粘土：可用于试制耐火砖，但需要处理含水率和烧制曲线。
- 劣质魔石：输出不稳定，不适合旧式高耗魔设备，但可作为低魔耗体系试验材料。
- 废旧魔导工坊：设备不完整，但可提供外壳、旧管路、铭文残件、工具和失败样本。
- 低阶人力：工匠、学徒、领民有经验但缺少统一记录和培训。

短缺资源：

- 现金。
- 粮食。
- 高品质魔石。
- 高阶法师。
- 标准化工具。
- 完整材料数据库。
- 合法认证渠道。

## 灰石领的社会状态

- 领民贫困，长期不信任原身莱恩。
- 旧贵族体面仍存在，但实际救不了人。
- 城堡破旧，男爵大厅仍象征家族尊严。
- 工坊半废弃，既是资产，也是负担。
- 旧暖炉停摆后，领民生存压力立刻显形。

## 第一卷中的灰石领变化

```text
破落领地
→ 旧炉停摆
→ 工坊重启
→ 灰炉一型勉强可用
→ 男爵大厅开放避寒
→ 贡献契约诞生
→ 行会调查令到来
```

# 05 旧魔导工业体系

## 旧体系不是愚蠢

旧魔导工业曾经有效。不能把旧体系写成全员愚蠢。

它的问题是：

```text
它太适配高魔时代，
却没有为低魔时代准备足够的低成本、可复制、可维护方案。
```

## 旧体系依赖

- 高魔环境补偿设备低效率。
- 高品质魔石掩盖材料缺陷。
- 大师经验代替标准记录。
- 家族秘方代替公开数据库。
- 行会认证代替公开测试。
- 贵族市场支付高维护成本。

## 末法后暴露的问题

- 魔石输出波动增大。
- 老设备吃魔严重。
- 高阶铭文维护成本高。
- 旧炉在劣质魔石下容易熄火或失控。
- 行会认证流程慢，费用高。
- 低阶魔法师被传统战斗评价体系淘汰。
- 平民和小领地无法承担贵族式魔导设备。

## 主角改造方向

主角不追求立刻恢复高魔时代辉煌，而是建立：

```text
低魔耗
低成本
可测试
可记录
可培训
可维护
可署名
可复制
```

的魔法工业雏形。

# 06 魔法能力工业化规则

## 总原则

传统魔法体系看：

```text
血统、天赋、爆发、上限、战斗表现、行会等级。
```

主角的工业体系看：

```text
稳定性、重复性、可记录性、岗位适配、误差范围、维护成本。
```

核心价值观：

```text
战斗看上限，工业看稳定。
天赋看血统，工序看可重复。
秘方看垄断，工程看记录。
```

## 五系工业用途

| 魔法类别 | 传统用途 | 工业用途 | 第一卷使用程度 |
|---|---|---|---|
| 火系 | 战斗、炼金、贵族取暖 | 控温、热处理、烧结、烘干、锅炉点火 | 核心使用，埃文负责控温 |
| 土系 | 筑墙、防御、土刺 | 粘土筛选、耐火砖成型、模具、夯实、矿石粗选 | 可点到，不展开成主线 |
| 风系 | 风刃、飞行、侦察 | 鼓风、通风、风选矿粉、烘干、气流测试 | 可作为后续扩展伏笔 |
| 水系 | 治疗、净水、战斗 | 冷却循环、水泵、洗矿、冷凝、清洗工序 | 第一卷不重点展开 |
| 铭文系 | 高阶法阵、认证、契约 | 报警、读数、检测、状态标记、简单控制回路 | 只用于旧炉封印、读数和认证压力 |

## 第一卷必须坚持的技术边界

- 火系弱火不能突然变强火。
- 埃文的价值来自波动小，不是爆发增强。
- 灰炉一型不能一次完美成功。
- 旧炉失效不是因为旧人愚蠢，而是旧设计不适应劣质魔石与低魔环境。
- 主角只能通过观察、测试、失败和记录逐步接近答案。
- 魔法仍有黑箱，主角不能完全解释全部原理。

# 07 法师行会

## 基本定位

法师行会是旧魔导工业认证权和安全审查权的掌握者。

它不是纯反派组织。

它同时具有两面性：

1. 确实积累了大量事故经验和魔导器安全标准。
2. 也确实垄断认证、维护秘方、收取费用，并压制低成本替代路线。

## 行会合理性

行会坚持认证，有其现实依据：

- 魔导器失控会伤人。
- 高阶铭文错误可能引发爆炸、倒灌、灼伤或魔力污染。
- 非专业贵族和工坊私改设备风险很高。
- 寒潮中大量领民集中使用灰炉，一旦出事故后果严重。

## 行会问题

- 认证费用高。
- 流程慢。
- 偏向旧设备和贵族客户。
- 评价体系重上限、轻稳定。
- 对低阶魔法师的岗位价值缺乏认知。
- 对公开实验记录不够敏感，更相信等级、师承和认证。

## 维克多的功能

维克多代表法师行会进入第一卷时，必须体现：

```text
他质疑灰炉，是因为安全、规程和责任，
不是因为他愚蠢、嫉妒或天生邪恶。
```

他与主角争的是验证体系：

```text
行会认证体系 vs 工程验证体系
权威许可 vs 公开数据
封闭经验 vs 可复现实验记录
```

# 08 商会与买断逻辑

## 基本定位

边境商会是旧市场体系的一部分，掌握现金、运输、渠道、债务和合同。

商会不是只会抢劫，而是通过合法合同完成控制。

## 商会合理性

商会能提供灰石领急需的现实资源：

- 现金。
- 债务延期。
- 税款周转。
- 原料采购。
- 销售渠道。
- 运输网络。
- 设备推广。

因此赫曼的买断合同必须足够诱人。

## 商会危险性

商会的危险不在于明显违法，而在于：

- 买断图纸。
- 买断制造权。
- 买断署名权或抹平参与者贡献。
- 将工坊成果变成商会垄断商品。
- 让埃文、工匠、领民重新变成无名劳动力。
- 复刻陆衡前世“成果被拿走”的伤口。

## 主角拒绝边界

主角不是拒绝商业合作。

主角拒绝的是：

```text
卖断署名。
卖断制造权。
卖断贡献记录。
卖断未来制度主权。
```

可以允许：

- 有限销售。
- 授权制造。
- 分成合作。
- 区域代理。
- 明确署名和责任的契约。

但第一卷先建立工坊贡献契约，不急着展开完整商业制度。

# 09 贵族、家臣与魔法教育门槛

## 贵族体系

灰石领仍处在贵族封地秩序下。

莱恩·格雷作为男爵，拥有名义上的：

- 领地管理权。
- 工坊资产处置权。
- 调用家臣和领民劳役的传统权力。
- 与税务官、商会、行会谈判的身份资格。

但他的现实权力很脆弱：

- 缺钱。
- 欠税。
- 领民不信。
- 工坊半废。
- 行会可质疑魔导器合法性。
- 商会可用债务施压。

## 家臣体系

巴伦不是普通老仆。

他是格雷家族世袭管家兼旧家臣，代表：

- 家族记忆。
- 旧贵族礼法。
- 家臣责任。
- 格雷家最后体面。
- 埃文获得魔法启蒙的合理来源。

## 魔法教育门槛

只有以下人群较可能获得魔法启蒙：

- 贵族子弟。
- 家臣家庭。
- 行会担保家庭。
- 富商或工坊主子弟。
- 少数被行会发现并投资的平民天赋者。

埃文作为巴伦孙子，能成为火系魔法学徒是合理的，因为巴伦是格雷家族世袭家臣。

## 第一卷冲突效果

这一设定制造三重压力：

1. 埃文被行会退回，不只是个人失败，也是巴伦家臣身份的羞耻。
2. 主角重新定义埃文价值，不只是技术发现，也是对旧评价体系的挑战。
3. 工坊贡献契约让低阶参与者获得署名，开始动摇贵族与行会垄断的荣誉结构。

# 10 灰炉一型

## 产品定位

灰炉一型是第一卷核心产品。

它不是神器，也不是完美工业设备。

它是：

```text
在灰石领资源极差、寒潮迫近、旧炉停摆、行会认证压力下，
用低成本试错勉强做出的低魔耗蓄热炉雏形。
```

## 技术目标

- 能使用劣质魔石。
- 对魔力波动有一定缓冲能力。
- 依赖低阶火系稳定控温，而非高阶火系爆发。
- 结构粗糙但可维护。
- 可记录炉温曲线。
- 可让领民熬过第一轮寒潮。

## 技术缺陷

- 炉芯砖会裂。
- 稳定性不足。
- 安全认证缺失。
- 对操作员依赖较高。
- 量产条件不足。
- 没有完整理论解释。
- 公开测试后会引来行会调查。

## 第一卷技术爽点

```text
旧体系判断劣质魔石无价值
→ 主角发现问题不是低，而是波动
→ 埃文弱火被重新定义为稳定控温能力
→ 失败记录逐步减少裂纹和熄火
→ 灰炉在寒潮夜撑住第一批领民
→ 公开测试中旧炉在劣质魔石下熄火
→ 灰炉一型证明低魔耗路线有价值
```

# 11 信息释放规则

## 第一卷允许释放的信息

第一卷可以释放：

- 旧暖炉停摆。
- 旧魔导器维修昂贵。
- 灰石领欠税。
- 领民冻病。
- 魔石质量下降。
- 行会认证重要。
- 私拆魔导器违法或违规。
- 火系学徒传统上看爆发力。
- 埃文火焰弱但波动小。
- 灰炉一型需要失败、记录和工序协作。

## 第一卷禁止释放的信息

第一卷不要释放：

- 末法时代真正历史原因。
- 高魔时代崩溃全貌。
- 帝国大战。
- 高阶神秘力量。
- 完整魔法理论。
- 终极工业蓝图。
- 主角未来完整制度设计。

## 信息出现方式

世界信息必须通过场景出现，不要直接百科讲解。

可使用方式：

| 信息类型 | 推荐出现方式 |
|---|---|
| 欠税、债务 | 账本、税务官通知、巴伦说明 |
| 旧炉问题 | 拆炉、封印记录、失败实验 |
| 魔石劣化 | 测试数据、熄火、波动曲线 |
| 领民困境 | 大厅避寒、冻病老人、饥饿孩子 |
| 行会认证 | 封印、维克多质疑、公开测试要求 |
| 商会买断 | 赫曼合同、债务压力、制造权条款 |
| 魔法教育门槛 | 埃文经历、巴伦羞愧、行会退回记录 |

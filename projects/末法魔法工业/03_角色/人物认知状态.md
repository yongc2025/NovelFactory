# 01 使用原则

人物认知状态用于控制“角色此时能知道什么”。

写章节卡、场景卡、正文草稿前，必须检查：

1. 本章人物有没有知道过早。
2. 本章人物有没有忘记已经知道的事。
3. 本章误解是否有来源。
4. 新判断是否来自正文中的观察、账本、失败实验、对话或行动。
5. 主角是否用前世知识直接跳过了异界信息获取过程。
6. 配角是否只为主角提供信息，而没有自己的误解和立场。

# 05 章节使用检查

每章写作前检查：

- 本章主角现在知道什么。
- 主角现在为什么行动。
- 本章新信息从哪里来：账本、旧物、环境、对话、失败实验、民生场景、专家解释。
- 本章谁产生了新误解。
- 本章谁纠正了旧误解。
- 本章是否有人知道了不该提前知道的信息。
- 本章认知变化是否带来行动变化。

# 06 更新规则

每完成一个章节卡或正文草稿后，必须更新：

1. 已知事实。
2. 新增误解。
3. 已纠正误解。
4. 仍不能提前知道的信息。
5. 关系认知变化。
6. 下一章需要承接的认知压力。

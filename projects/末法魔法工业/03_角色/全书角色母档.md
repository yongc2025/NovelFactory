# 01 全书角色管理原则

长篇角色不能每卷重置。

每进入新卷前，必须检查：

1. 上一卷结束后，角色已经变成什么。
2. 本卷世界压力升级到了哪一层。
3. 旧角色是否还能承载新压力。
4. 是否需要新增角色来承接新制度、新阵营、新矛盾。
5. 本卷结束时，角色关系结构发生了什么不可逆变化。

角色成长必须遵守：

```text
旧伤口不重复消费。
旧成长不回退。
新压力必须逼出新选择。
新选择必须带来新代价。
```

# 04 第一卷五人组长期去向

## 陆衡 / 莱恩

第一卷结束状态：

```text
不完全接受自己就是莱恩，但愿意用莱恩之名保护工坊成果。
```

长期问题：

```text
他不能永远用“我不是莱恩”回避责任。
但也不能突然变成天生领主。
```

后续成长方向：

- 第2卷：学会授权。
- 第3卷：学会与军需体系交易。
- 第4卷：学会面对王室合法性。
- 中期：面对自己体系可能形成新垄断。
- 后期：从成果保护者变成秩序维护者。

## 巴伦

第一卷结束状态：

```text
初步承认变化后的莱恩正在承担格雷家过去没做到的事。
```

长期核心：

```text
从守旧体面，到把新制度翻译成新的格雷家礼法。
```

后续成长方向：

- 第2卷：从旧管家转为授权契约见证人。
- 第3卷：面对军需体系进入灰石领，守住格雷家的自主边界。
- 第4卷：王室特许带来新体面，也带来新控制，他要重新理解“忠诚”。
- 中后期：可能成为灰石制度礼法的象征，而非一线管家。

## 埃文·巴伦

第一卷结束状态：

```text
从失败学徒成为一号炉温控制员，获得署名、岗位和身份尊严。
```

长期核心：

```text
从“我不是废物”，到“我能教别人不再被判定为废物”。
```

后续成长方向：

- 第2卷：开始教别人控温，面对教学失败。
- 第3卷：进入军需试制，面对军方对稳定性的严苛要求。
- 中期：成为技师学校早期核心人物。
- 后期：代表低阶技师体系，与行会学徒制度正面对照。

## 维克多

第一卷结束状态：

```text
无法否认灰炉数据，但仍保留行会调查立场。
```

长期核心：

```text
从“认证高于现场”，到“认证必须吸收可复现实验数据”。
```

后续成长方向：

- 第2卷：推动行会基层调查。
- 第3卷：面对军需署对灰炉路线的兴趣，行会权威被绕开。
- 中期：在行会内部陷入保守派与实用派之间。
- 后期：可发展为认证体系改革接口，或者成为坚持底线的制度对手。

## 赫曼

第一卷结束状态：

```text
买断失败，但意识到灰炉不是普通商品，陆衡不是普通缺钱贵族。
```

长期核心：

```text
从“我要买断技术”，到“我要证明没有市场控制，技术活不下去”。
```

后续成长方向：

- 第2卷：尝试独家代理、渠道控制、维修件捆绑。
- 第3卷：军需体系进入后，商会渠道被挤压。
- 第4卷：王室财政派注意灰炉后，商会必须重新定位。
- 中后期：可成为复杂合作方，也可能成为灰炉体系商业侧最大对手。

# 05 后续新增角色规则

新增角色必须满足至少一项：

1. 承接新世界压力。
2. 挑战主角当前错误信念。
3. 与旧角色产生横向关系。
4. 逼主角付出新的制度代价。
5. 让灰石体系暴露新风险。

禁止新增只负责热闹、恋爱、打脸或说明设定的角色。

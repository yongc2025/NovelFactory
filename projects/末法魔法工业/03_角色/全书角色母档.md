# 01 全书角色管理原则

长篇角色不能每卷重置。

每进入新卷前，必须检查：

1. 上一卷结束后，角色已经变成什么。
2. 本卷世界压力升级到了哪一层。
3. 旧角色是否还能承载新压力。
4. 是否需要新增角色来承接新制度、新阵营、新矛盾。
5. 本卷结束时，角色关系结构发生了什么不可逆变化。

角色成长必须遵守：

```text
旧伤口不重复消费。
旧成长不回退。
新压力必须逼出新选择。
新选择必须带来新代价。
```

## 1.1 三层角色管理

```text
全书角色母档

```

本文件只管全书战略，不承担章节细节。

# 04 第一卷五人组长期去向

## 陆衡 / 莱恩

第一卷结束状态：

```text
不完全接受自己就是莱恩，但愿意用莱恩之名保护工坊成果。
```

长期问题：

```text
他不能永远用“我不是莱恩”回避责任。
但也不能突然变成天生领主。
```

后续成长方向：

- 第2卷：学会授权。
- 第3卷：学会与军需体系交易。
- 第4卷：学会面对王室合法性。
- 中期：面对自己体系可能形成新垄断。
- 后期：从成果保护者变成秩序维护者。

## 巴伦

第一卷结束状态：

```text
初步承认变化后的莱恩正在承担格雷家过去没做到的事。
```

长期核心：

```text
从守旧体面，到把新制度翻译成新的格雷家礼法。
```

后续成长方向：

- 第2卷：从旧管家转为授权契约见证人。
- 第3卷：面对军需体系进入灰石领，守住格雷家的自主边界。
- 第4卷：王室特许带来新体面，也带来新控制，他要重新理解“忠诚”。
- 中后期：可能成为灰石制度礼法的象征，而非一线管家。

## 埃文·巴伦

第一卷结束状态：

```text
从失败学徒成为一号炉温控制员，获得署名、岗位和身份尊严。
```

长期核心：

```text
从“我不是废物”，到“我能教别人不再被判定为废物”。
```

后续成长方向：

- 第2卷：开始教别人控温，面对教学失败。
- 第3卷：进入军需试制，面对军方对稳定性的严苛要求。
- 中期：成为技师学校早期核心人物。
- 后期：代表低阶技师体系，与行会学徒制度正面对照。

## 维克多

第一卷结束状态：

```text
无法否认灰炉数据，但仍保留行会调查立场。
```

长期核心：

```text
从“认证高于现场”，到“认证必须吸收可复现实验数据”。
```

后续成长方向：

- 第2卷：推动行会基层调查。
- 第3卷：面对军需署对灰炉路线的兴趣，行会权威被绕开。
- 中期：在行会内部陷入保守派与实用派之间。
- 后期：可发展为认证体系改革接口，或者成为坚持底线的制度对手。

## 赫曼

第一卷结束状态：

```text
买断失败，但意识到灰炉不是普通商品，陆衡不是普通缺钱贵族。
```

长期核心：

```text
从“我要买断技术”，到“我要证明没有市场控制，技术活不下去”。
```

后续成长方向：

- 第2卷：尝试独家代理、渠道控制、维修件捆绑。
- 第3卷：军需体系进入后，商会渠道被挤压。
- 第4卷：王室财政派注意灰炉后，商会必须重新定位。
- 中后期：可成为复杂合作方，也可能成为灰炉体系商业侧最大对手。

# 05 后续新增角色规则

新增角色必须满足至少一项：

1. 承接新世界压力。
2. 挑战主角当前错误信念。
3. 与旧角色产生横向关系。
4. 逼主角付出新的制度代价。
5. 让灰石体系暴露新风险。

禁止新增只负责热闹、恋爱、打脸或说明设定的角色。

# 06 后续关键角色占位

## 北境军需署代表

- 所属阵营：北境军需署
- 建议登场：第3卷左右
- 核心功能：强机构保护伞，同时也是军用优先、备案审查和供应链控制压力。
- 对主角的挑战：灰炉如果进入军需体系，民用供暖是否要让位于军用订单。
- 角色底层逻辑：他不关心理想，关心冻伤率、运输损耗、哨塔存活率和军需可靠性。
- 禁止写法：不能写成单纯欣赏主角的伯乐。

## 王室财政派代表

- 所属阵营：王室财政派
- 建议登场：第4卷左右
- 核心功能：合法性、税收、特许权、备案和国家控制。
- 对主角的挑战：王室可以保护灰石体系，也可以收编灰石体系。
- 角色底层逻辑：国家不允许关键工业资产完全脱离财政和法务监管。
- 禁止写法：不能写成只会贪税的官僚。

## 行会改革派代表

- 所属阵营：法师行会内部实用派 / 改革派
- 建议登场：中期
- 核心功能：承认数据价值，但质疑主角的新体系是否会变成另一种垄断。
- 对主角的挑战：如果旧行会封闭，那么灰石标准是否也可能封闭。
- 角色底层逻辑：改革不是投降，行会必须保留专业审查权。
- 禁止写法：不能写成一见主角就跪服的技术迷弟。

## 大贵族吞并者

- 所属阵营：北境或王国大贵族封地集团
- 建议登场：中后期
- 核心功能：旧封地秩序对灰炉走廊的吞并压力。
- 对主角的挑战：灰石领掌握走廊节点，威胁传统封地收益与军道控制权。
- 角色底层逻辑：大领地认为关键节点必须由更高爵位、更大军力、更久家族掌控。
- 禁止写法：不能写成无脑贵族恶霸。

## 邻国技术争夺者

- 所属阵营：邻国军需 / 商会 / 情报系统
- 建议登场：后期
- 核心功能：低魔工业国际化争夺、技术防扩散、寒线战争压力。
- 对主角的挑战：灰炉体系一旦成为战略资产，就不可能只被本国争夺。
- 角色底层逻辑：邻国不是偷东西，而是在争夺边境战争中的生存优势。
- 禁止写法：不能写成低级间谍小丑。

## 长线女性角色

- 当前状态：待定，不强行设女主。
- 可选方向：本地女性工匠、北境军需官、王室财政派女官、行会改革派女铭文师。
- 设计原则：必须承接结构功能，不能只承担恋爱、吐槽、照顾主角或美化群像。
- 推荐优先级：本地女性工匠或军需体系角色。

# 08 全书角色审校清单

写后续卷前逐项检查：

1. 本卷是否重复了上一卷已经解决的角色问题。
2. 主角伤口是否升级为新问题，而不是原地反复疼。
3. 巴伦、埃文、维克多、赫曼是否有新功能，或合理退居二线。
4. 新角色是否承接新世界压力。
5. 新角色是否与旧角色有横向关系。
6. 新制度是否带来新代价。
7. 灰石体系是否暴露新的风险。
8. 普通人是否仍能体现制度变化的基层结果。
9. 主角是否正在从成果保护者走向秩序维护者。
10. 终局寒岭侯是否被一步步压出来，而不是突然封爵。

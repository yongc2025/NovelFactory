# 灵感池 v0.2：身份线修订

## 文件用途

本文件是 `灵感池.md` 的正式补丁，专门修订第一卷中陆衡 / 莱恩身份接受线的处理方式。

后续更新主灵感池时，应以本文件为准，将旧版“三阶段身份接受”替换为新版“四阶段暗线身份节奏”。

## 修订原因

旧版设计：

```text
第 1-10 章：不承认身份
第 11-20 章：被迫使用身份
第 21-30 章：主动保护身份背后的责任
```

问题：

```text
这个设计有质感，但如果直接执行，会显得不太符合男频常规节奏。
前 10 章“不承认身份”容易被写成心理拖延，压过灰炉、寒潮、领地危机和魔法工业化主线。
```

修订判断：

```text
身份接受可以慢。
但主角使用身份必须早。
外部行动必须快。
心理承认可以作为暗线慢慢推进。
```

## 第一卷明暗线

第一卷明线：

```text
灰石领如何活过寒潮，灰炉一型如何诞生。
```

第一卷暗线：

```text
陆衡如何从借用莱恩身份求生，到愿意用莱恩这个名字承担责任。
```

身份线必须做暗线，不要压过灰炉、寒潮、领地危机和魔法工业化主线。

## 新版四阶段身份节奏

### 第一阶段：身份惊醒，求生伪装

对应章节：第 1-3 章。

主角心理：

```text
我不是莱恩。
但在弄清楚这里之前，我必须让他们以为我是莱恩。
```

表现方式：

- 别人叫“莱恩少爷”时，他会慢半拍。
- 他看镜子里的脸会觉得陌生。
- 他把身体称为“这具身体”。
- 原身记忆像别人的资料，不像自己的亲身经历。
- 巴伦试探时，他用失忆或高烧后遗症遮掩。
- 他不主动说“我是莱恩”。

关键词：伪装求生。

### 第二阶段：临时代理，先救眼前

对应章节：第 4-10 章。

主角心理：

```text
我不是莱恩，但这个身份能让我查账、调人、修炉。
```

表现方式：

- 他把男爵身份当工具，不当自我。
- 他会确认自己是否有权看账、调人、开仓、进工坊。
- 他使用莱恩身份解决现实问题，但内心仍称自己陆衡。
- 他会用工程问题转移身份追问。
- 他下命令时语气偏冷静、试探，不是自然的贵族命令。

关键词：临时代理。

### 第三阶段：责任绑定，被人和事拖住

对应章节：第 11-20 章。

主角心理：

```text
我不是莱恩，但这些人正在因为这个名字看着我。
```

表现方式：

- 他第一次以莱恩·格雷的名义签税务延期或工坊文件时，会产生强烈不适。
- 签名触发前世署名被夺的伤口。
- 他开始保护埃文、工匠、领民，但嘴上仍不承认自己就是莱恩。
- 巴伦不再只怀疑，而是开始观察这个变化后的莱恩是否值得相信。
- 他开放男爵大厅给领民避寒，实际承担了领主责任，但心理上仍没有完全认同身份。

关键词：被迫承担。

### 第四阶段：价值认领，用莱恩之名立规矩

对应章节：第 21-30 章。

主角心理：

```text
我也许不是原来的莱恩，但我不能让这个名字卖掉他们的成果。
```

表现方式：

- 灰炉救人后，商会提出买断图纸、制造权和署名权。
- 买断合同复刻陆衡前世成果被夺的伤口。
- 主角拒绝卖断制造权和署名权。
- 他建立工坊贡献契约，规定参与者署名、收益、责任和记录。
- 他第一次主动用莱恩·格雷这个名字保护他人的劳动成果。

关键词：价值认领。

## 身份线表现细节

### 1. 名字反应慢半拍

别人叫他“莱恩少爷”时，他不会立刻反应。

### 2. 叙述视角仍称他为陆衡

前期叙述应优先使用“陆衡”，而不是直接称他为“莱恩”。

### 3. 身体陌生感

他可以把身体称为“这具身体”，但不能滥用。

### 4. 原身记忆像资料

原身记忆可以出现，但情感归属要隔一层。

### 5. 签名迟疑

第一次签下“莱恩·格雷”必须是强节点，因为这会触发前世署名被夺的伤口。

### 6. 用工程问题逃避身份问题

巴伦追问他变化时，他可以转向暖炉、账本、工坊、魔石等现实问题。

### 7. 对贵族礼仪不适

徽章、主位、家族荣誉、老主人记忆都可以成为身份排斥的细节。

## 巴伦合理接受逻辑

巴伦不接受“陆衡”，也不知道魂穿。

巴伦理解为：

```text
少爷病后性情大变。
少爷也许失忆。
少爷也许在濒死时受到了祖灵启示。
少爷也许已经不是过去那个软弱的莱恩。
```

巴伦接受的不是魂穿真相，而是：

```text
这个变化后的莱恩，至少正在承担格雷家族过去没做到的事。
```

巴伦逐步接受的证据：

1. 主角没有卷钱跑路，也没有卖掉工坊。
2. 主角没有把埃文当废物，而是重新定义了埃文的弱火价值。
3. 主角在实验记录本上写下每个参与者名字，没有独占成果。
4. 灰炉救下领民后，主角仍拒绝商会买断署名和制造权。

## 第一卷核心身份态度四句话

```text
第 1-3 章：我不是莱恩，但我不能露馅。
第 4-10 章：我不是莱恩，但这个身份能让我查账、调人、修炉。
第 11-20 章：我不是莱恩，但这些人正在因为这个名字看着我。
第 21-30 章：我也许不是原来的莱恩，但我不能让这个名字卖掉他们的成果。
```

## 写作禁区

- 不要反复写“我不是莱恩”式心理独白。
- 不要让主角因为不接受身份而停止行动。
- 不要让身份线压过灰炉和寒潮主线。
- 不要让巴伦无脑接受主角变化。
- 不要让主角过早说“我是灰石领领主”。
- 不要在第一卷让主角完全完成身份融合。

## 对主灵感池的替换要求

主灵感池中的旧段落：

```text
第一卷身份接受分三次完成：行动承认、责任承认、价值承认。
```

应替换为：

```text
第一卷身份线做暗线，分四阶段推进：身份惊醒，求生伪装；临时代理，先救眼前；责任绑定，被人和事拖住；价值认领，用莱恩之名立规矩。
```

主灵感池中的旧标题：

```text
第一卷身份接受节奏
```

应替换为：

```text
第一卷身份线：暗线处理
```

# 01 已确认核心 canon

## 项目基本 canon

- 主角前世名：陆衡
- 异界身份：莱恩·格雷
- 领地：灰石领
- 第一卷：灰炉与寒潮
- 第一项产品：灰炉一型，低魔耗蓄热炉
- 火系魔法学徒：埃文·巴伦，男性，巴伦的孙子
- 巴伦身份：格雷家族世袭管家兼旧家臣，不是普通仆人
- 主角接受身份需要慢热推进，不能一穿越就接受莱恩身份

## 世界背景 canon

- 末法时代不是自然缓慢衰退，而是高魔时代战争导致魔力基础设施崩坏后的断代社会。
- 高魔时代魔法工业依赖地脉节点、魔晶炉群、地脉锚点、调律塔、城市魔网、行会认证接入体系。
- 距今约九十六年前，烬日战争末期，各方攻击彼此的魔力基础设施，并动用禁忌级战略魔导器，导致地脉锚点崩坏、魔力潮汐失衡、魔石杂质化。
- 世界并未失去魔法，但失去了高魔工业所需的稳定、高纯、高浓度魔力输入。
- 当前世界状态是：后高魔遗产社会 + 手工业生产 + 行会认证维护体系 + 边境工业退化。

## 工业路线 canon

陆衡穿越后不直接从蒸汽机开始。

第一卷的正确工程起点是生存热工：

```text
灰炉一型
→ 耐火材料
→ 控温岗位
→ 劣质魔石波动测试
→ 炉温记录
→ 低魔耗燃烧
→ 工坊贡献契约雏形
```

长篇工业演进路线：

```text
生存热工
→ 工坊标准化
→ 金工与机械基础
→ 低压动力
→ 交通物流
→ 城镇基础设施
→ 铭文仪表与控制
→ 低魔力网络
```

## 六系岗位化 canon

- 火系：热工岗位，控温、热处理、烧结、烘干、锅炉点火。
- 土系：土石 / 陶工 / 基建岗位，成型、夯实、矿石粗选、模具、地基。
- 金系：金工 / 金属材料岗位，去杂、锻造辅助、管件、轴承、裂纹检测、标准件。
- 风系：流体 / 气动岗位，鼓风、通风、风选、减阻、交通助推。
- 水系：循环 / 冷却 / 清洗岗位，冷却循环、水泵、洗矿、冷凝、排水。
- 铭文系：仪表 / 控制 / 认证岗位，读数、报警、联锁、记录、防篡改、认证。

第一卷只重点使用火系、土系、铭文系。金系、风系、水系只作伏笔或辅助，不抢灰炉主线。

# 02 当前核心判断

这不是普通科技碾压异界，而是魔法能力工业化。

主角的现代知识提供的是方法：观察、建模、测试、失败、记录、改进。

主角不能瞬间理解魔法，也不能马上被巴伦、埃文、领民和工匠接受。

旧体系不是愚蠢，而是过时。法师行会、贵族秘方、商会渠道都有合理性，但合在一起形成了末法时代的低成本创新阻力。

---

# 03 第一卷重点

第一卷不写帝国大战，不写高级魔法理论，不写末法历史真相。

第一卷只聚焦：

```text
陆衡如何在不承认自己是莱恩的情况下，被寒潮、债务、领民和巴伦一步步推着承担领主责任，最终用灰炉一型让灰石领活过寒潮，并第一次用莱恩·格雷的名字保护技术成果。
```

第一卷四项任务：

1. 生存热工：灰炉一型从失败到勉强可用。
2. 身份暗线：陆衡从否认身份到价值层面使用莱恩之名。
3. 人物重估：埃文弱火从失败天赋变成控温岗位。
4. 制度雏形：工坊贡献契约保护署名、收益、责任和记录。

# 04 第一卷信息释放禁区

第一卷只能释放末法现象，不揭示完整历史真相。

## 允许释放

- 旧暖炉停摆。
- 魔石质量下降。
- 行会维修昂贵。
- 旧炉在劣质魔石下不稳定。
- 灰石领撑不过寒潮。
- 低阶火系学徒被旧评价体系淘汰。
- 法师行会认证很重要。
- 商会可以提供现金和渠道，但会要求买断。

## 禁止释放

- 烬日战争全貌。
- 地脉锚点崩坏细节。
- 调律塔和城市魔网结构。
- 禁忌战略魔导器真相。
- 高魔时代产业体系全貌。
- 低魔力网络的未来蓝图。
- 帝国大战。
- 高阶神秘力量。
- 完整魔法理论。

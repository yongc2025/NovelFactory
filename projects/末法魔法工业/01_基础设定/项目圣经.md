# 01 一句话故事

前世被科研体系压榨并夺走成果的陆衡，魂穿成末法时代破落男爵莱恩·格雷，在寒潮、债务、行会质疑、领民不信任和自身身份错位中，用现代材料学与机械工程方法，把低阶魔法能力改造成工业岗位，从第一台灰炉开始建立新的魔法工业秩序。

---

# 02 核心命题

真正改变世界的不是某个天才个人，而是能让知识被验证、被传承、被署名、被保护，并让普通人参与创造的体系。

# 03 读者承诺

## 表层读者承诺

- 看现代工程科研人用实验、材料、机械结构和生产流程改造末法魔法世界。
- 看破落领地从濒临崩溃到逐步建立工业体系。
- 看低阶魔法师、工匠、领民被重新定义价值。
- 看旧贵族、法师行会、商会垄断被新工业秩序一步步冲击。
- 看第一台灰炉如何从失败、裂损、质疑中成为灰石领活过寒潮的现实工具。

## 深层读者承诺

- 一个前世被夺走成果的人，在新世界建立保护劳动成果的制度。
- 一个不承认自己是领主的人，被现实、责任和人心一步步推向新的身份。
- 一个末法世界不是靠神迹复兴，而是靠测量、记录、标准、工序、教育和契约重建。
- 一个断代世界不是复原旧高魔辉煌，而是建立适合低魔环境的新工业网络。

# 04 主角设定

## 前世：陆衡

- 出身：普通农村家庭。
- 专业：材料学与机械制造方向。
- 职业：大型机械企业研发人员。
- 能力：材料失效分析、热处理、耐火材料、机械结构、工艺改造、质量控制、实验记录。
- 伤口：长期熬夜加班，成果刚出就被抢署名，最终意外身亡。
- 性格后果：不信权威头衔，不信秘方垄断，对契约、署名和实验记录极度敏感。

## 今生：莱恩·格雷

- 身份：灰石领破落男爵。
- 资产：破旧城堡、老管家巴伦、半废弃魔导工坊、旧暖炉、债务、欠税、贫困领民。
- 起点：冬季前夕，寒潮将至，旧暖炉停摆，税务官三天后抵达。
- 心理状态：行动上被迫扮演莱恩，心理上仍认为自己是陆衡。

## 主角欲望

### 外部欲望

让灰石领活过第一场寒潮。

### 中期欲望

建立一套不依赖高阶法师、贵族秘方和行会认证的低魔耗魔法工业体系。

### 深层欲望

建立一个不会夺走劳动成果、能让参与者被署名和被保护的新制度。

## 主角能力边界

主角的专业不是外挂，而是方法。

```text
观察 → 建模 → 测试 → 失败 → 记录 → 改进
```

主角不应马上做到：

- 完整理解魔法理论。
- 解释高魔时代历史真相。
- 绕过实验失败。
- 直接制造可用蒸汽机。
- 直接建立电气工业。
- 凭空获得完整工业体系。
- 让所有配角立刻相信他。

# 05 世界背景核心 canon

这是一个进入末法时代的魔法工业世界。

高魔时代曾经拥有辉煌的魔导工业。其基础不是单个强大法师，而是覆盖大陆的魔力基础设施：地脉节点、魔晶炉群、地脉锚点、调律塔、城市魔网、行会认证接入体系。

距今约九十六年前，烬日战争末期，各国、贵族集团和法师势力攻击彼此的魔力基础设施，并动用禁忌级战略魔导器，导致地脉锚点崩坏、魔力潮汐失衡、魔石杂质化。

世界并未失去魔法，但失去了高魔工业所需的稳定、高纯、高浓度魔力输入。

```text
过去：高浓度、高纯度、高稳定性的魔力输入。
现在：低浓度、高波动、高杂质的魔力输入。
```

旧魔导工业因此没有立刻消失，而是进入长期失配和退化：

- 魔力浓度下降。
- 魔石杂质增加。
- 高阶法师和底层魔导工程人才断代。
- 旧魔导设备频繁失效。
- 维护成本越来越高。
- 平民被重新挤出魔法工业消费体系。
- 贵族、法师行会和商会仍守着旧秘方、认证权和供应链。

---

# 06 传统魔法工业的问题

旧体系不是愚蠢，而是过时。

它曾经依赖：

- 高魔环境。
- 高品质魔石。
- 地脉节点和城市魔网。
- 大师经验。
- 家族秘方。
- 行会认证。
- 贵族垄断市场。

末法后暴露出：

- 能源输入条件失配。
- 材料和魔石品质退化。
- 底层研发人才断代。
- 战争后知识被炸毁、封存、污染、垄断。
- 缺少标准化测试。
- 缺少材料数据库。
- 缺少工艺记录。
- 缺少批量培训体系。
- 缺少低成本替代方案。
- 无法重新定义低阶魔法能力。

# 07 主角穿越后的工业现状

陆衡穿越时，这个世界不是正常中世纪，也不是标准蒸汽前夜，而是：

```text
后高魔遗产社会
+ 手工业生产
+ 行会认证维护体系
+ 边境工业退化
```

上层世界仍保有少量昂贵旧魔导设备，如城市供暖、魔导水泵、贵族温室、浮空运输残线、高价通讯塔和行会认证工坊。

边境小领地已经退化到手工锻造、人力搬运、木炭和劣质魔石混用、旧设备靠拆零件续命、工坊缺少标准件、账本混乱、领民取暖困难的状态。

灰石领旧暖炉停摆，不是因为魔法不存在，而是因为旧设备无法适应劣质魔石和低魔环境。

---

# 08 主角工业路线 canon

陆衡穿越后不直接从蒸汽机开始。

蒸汽动力需要可靠锅炉、耐压材料、密封件、阀门、管道、压力表、安全阀、轴承、加工精度和维修体系。灰石领开局没有这些条件。

第一卷的正确工程起点是生存热工：

```text
灰炉一型
→ 耐火材料
→ 控温岗位
→ 劣质魔石波动测试
→ 炉温记录
→ 低魔耗燃烧
→ 工坊贡献契约雏形
```

长篇工业演进路线：

```text
生存热工
→ 工坊标准化
→ 金工与机械基础
→ 低压动力
→ 交通物流
→ 城镇基础设施
→ 铭文仪表与控制
→ 低魔力网络
```

这条路线不是复原旧高魔魔网，而是建立低压、低耗、分布式、可维护、可追责的新魔法工业网络。

# 09 魔法能力工业化原则

主角不会把魔法当成神秘天赋，而是拆成可记录、可训练、可替换、可协作的工序能力。

核心价值观：

```text
战斗看上限，工业看稳定。
天赋看血统，工序看可重复。
秘方看垄断，工程看记录。
魔法能力不是职业身份，而是工序能力。
```

六系岗位化：

```text
火系：热工岗位，控温、热处理、烧结、烘干、锅炉点火。
土系：土石 / 陶工 / 基建岗位，成型、夯实、矿石粗选、模具、地基。
金系：金工 / 金属材料岗位，去杂、锻造辅助、管件、轴承、裂纹检测、标准件。
风系：流体 / 气动岗位，鼓风、通风、风选、减阻、交通助推。
水系：循环 / 冷却 / 清洗岗位，冷却循环、水泵、洗矿、冷凝、排水。
铭文系：仪表 / 控制 / 认证岗位，读数、报警、联锁、记录、防篡改、认证。
```

第一卷主线只重点使用火系、土系、铭文系。金系、风系、水系可作为伏笔或辅助，不抢灰炉主线。

# 10 关键配角

## 巴伦

- 身份：格雷家族世袭管家兼旧家臣。
- 核心欲望：守住格雷家最后的尊严。
- 初始态度：怀疑主角变化，反对主角进入工坊。
- 变化方向：从守旧贵族体面，到见证新工坊契约。
- 写作重点：巴伦不能无脑信任主角。他接受的不是“陆衡”，而是“这个变化后的莱恩至少正在承担格雷家过去没做到的事”。

## 埃文·巴伦

- 身份：巴伦的孙子，低阶火系魔法学徒。
- 性别：男性。
- 合理性：家臣家庭有资格接受魔法启蒙。
- 传统评价：火焰弱、爆发低、无战斗价值，被行会退回。
- 主角评价：火焰弱但稳定，适合作为灰炉控温员。
- 弧光：失败学徒 → 一号炉温控制员 → 第一代魔法工业技师。
- 写作重点：埃文的爽点不是弱火突然变强，而是弱火被重新定义为稳定控温价值。

## 维克多

- 身份：法师行会认证法师。
- 立场：魔导器危险，必须认证，否则可能害人。
- 功能：旧秩序代表，不是小丑反派。
- 写作重点：维克多的合理性来自安全、规程和事故责任。他与主角争的是行会认证体系 vs 工程验证体系。

## 赫曼

- 身份：边境商会代表。
- 立场：成果可以被合同买断，技术可以变成垄断商品。
- 功能：前世伤口镜像。
- 写作重点：赫曼不是低级奸商。他的合同必须合法、诱人、危险，能提供灰石领急需现金，同时要求图纸、制造权和署名控制。

# 11 第一卷目标

第一卷《灰炉与寒潮》只解决一件事：灰石领如何活过寒潮。

技术线：灰炉一型从失败到可用。

人物线：陆衡从否认身份，到被迫使用莱恩身份，再到第一次主动用这个身份保护他人劳动成果。

制度线：从个人试错到工坊贡献契约。

外部压力：寒潮、欠税、商会买断、行会调查。

第一卷不写帝国大战，不写高级魔法理论，不写末法历史真相。

第一卷只呈现末法现象：旧暖炉停摆、魔石变差、维修昂贵、行会认证慢、领民用不起旧设备、灰石领撑不过寒潮。

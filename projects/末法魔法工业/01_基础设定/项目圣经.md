# 01 一句话故事

前世被科研体系压榨并夺走成果的材料工程博士生陆衡，魂穿成末法时代破落男爵莱恩·格雷，在寒潮、债务、行会质疑、领民不信任、国家机器收编和邻国争夺中，从第一台低魔耗灰炉开始，把被旧体系淘汰的低阶魔法能力改造成可记录、可培训、可维护、可署名、可保护的工业岗位，最终把灰石领从破落小封地建设成北境低魔工业走廊核心，并以寒岭侯身份确立灰炉纪元。

# 02 核心命题

真正改变世界的不是某个天才个人，而是能让知识被验证、被传承、被署名、被保护，并让普通人参与创造的体系。

但本书必须同时承认：

```text
契约只能保护愿意守规则的人。
工业武装和利益共同体，才能逼不愿守规则的人计算代价。
```

因此，陆衡最终要建立的不是一张理想化契约，而是一套拥有产能、财政、武力、制度、联盟和战略依赖的低魔工业共同体。

# 03 读者承诺

## 表层读者承诺

- 看现代材料与机械工程方法拆解旧魔法工业。
- 看破落边境领地从第一台灰炉开始翻身。
- 看魔法学徒、低阶法师、工匠、矿工、领民被重新定义为工业岗位。
- 看旧贵族、法师行会、商会垄断被低魔工业体系逐步挑战。
- 看灰石领从寒潮求生，到灰炉走廊，到北境低魔工业共同体。
- 看主角从灰石男爵成长为寒岭侯，但不走公爵、亲王、建国或王位继承路线。

## 深层读者承诺

- 一个前世被夺走成果的人，在新世界建立保护劳动成果的制度。
- 一个不承认自己是领主的人，被现实、责任、人心、战争和权力一步步推向新的身份。
- 一个末法世界不是靠神迹复兴，而是靠测量、记录、标准、工序、教育、财政、武装和政治交易重建。
- 一个断代世界不是复原旧高魔辉煌，而是建立适合低魔环境的新工业网络。
- 一个小贵族不是靠道德说服强权，而是在军需、财政、边境安全和强权博弈中为新制度争取生存空间。

- 身体年龄：19 岁。
- 起点身份：灰石男爵。
- 起点资产：破旧城堡、老管家巴伦、半废弃魔导工坊、旧暖炉、债务、欠税、贫困领民。

# 06 世界背景核心 canon

这是一个进入末法时代的魔法工业世界。

高魔时代曾经拥有辉煌的魔导工业。其基础不是单个强大法师，而是覆盖大陆的魔力基础设施：地脉节点、魔晶炉群、地脉锚点、调律塔、城市魔网、行会认证接入体系。

距今约九十六年前，烬日战争末期，各国、贵族集团和法师势力攻击彼此的魔力基础设施，并动用禁忌级战略魔导器，导致地脉锚点崩坏、魔力潮汐失衡、魔石杂质化。

世界并未失去魔法，但失去了高魔工业所需的稳定、高纯、高浓度魔力输入。

```text
过去：高浓度、高纯度、高稳定性的魔力输入。
现在：低浓度、高波动、高杂质的魔力输入。
```

旧魔导工业因此没有立刻消失，而是进入长期失配和退化：

- 魔力浓度下降。
- 魔石杂质增加。
- 高阶晋升环境崩坏。
- 高阶魔导工程人才断代。
- 旧魔导设备频繁失效。
- 维护成本越来越高。
- 平民被重新挤出魔法工业消费体系。
- 贵族、法师行会和商会仍守着旧秘方、认证权和供应链。

# 07 魔法师等级边界 canon

## 高魔时代完整等级

高魔时代的完整魔法等级为：

```text
魔法学徒
→ 低阶法师
→ 中阶法师
→ 高阶法师
→ 大法师
→ 魔导师
→ 法神
```

高魔时代的城市魔网、调律塔、地脉锚点、战略魔导器和跨城魔力网络，依赖高阶以上层级真实存在。

## 末法时代公开等级

末法时代公开社会的实际等级只剩：

```text
魔法学徒
→ 低阶法师
→ 中阶法师
```

公开社会的实际上限是中阶法师。

普通魔法师大多停留在低阶法师，优秀者可晋升中阶法师。

经营阶层、地方认证法师、军需魔导官、贵族工坊负责人、大型商会魔导主管，通常以中阶法师为上限。

## 高阶法师

高阶法师在末法时代已经属于国家战略级存在。

```text
可能真实存在，但从未被公开证实。
```

若存在，也只可能掌握在：

- 王室最高层。
- 法师行会总部。
- 国家军团核心。
- 顶级大贵族。
- 最高级封印项目。
- 战略魔导设施。

高阶法师不参与日常社会运转，不处理地方工坊、商会买断、边境供暖或普通认证争端。

高阶法师一旦公开出手，就是国家级战略威慑暴露，会引发政治、军事和邻国层面的连锁反应。

## 大法师、魔导师、法神

大法师、魔导师和法神在末法时代已彻底消失。

它们只存在于：

- 高魔时代遗迹。
- 禁忌档案。
- 王室秘闻。
- 行会最高文献。
- 破碎调律塔铭文。
- 民间传说和神话争议。

不可作为当代活跃人物进入主线。

## 主角路线意义

陆衡的低魔工业路线不试图对抗高阶法师的个人上限。

他真正争夺的是：

```text
魔法学徒
低阶法师
中阶实用派
```

他的体系核心是：

```text
把学徒变成操作员；
把低阶法师变成岗位技师；
把中阶法师中的实用派变成认证和管理合作者。
```

这条规则是全书战力平衡底线。

# 08 传统魔法工业的问题

旧体系不是愚蠢，而是过时。

它曾经依赖：

- 高魔环境。
- 高品质魔石。
- 地脉节点和城市魔网。
- 大师经验。
- 家族秘方。
- 行会认证。
- 贵族垄断市场。
- 中阶认证法师维持公开社会运行。
- 高阶以上传说维持王室和行会总部的战略威慑。

末法后暴露出：

- 能源输入条件失配。
- 材料和魔石品质退化。
- 高阶晋升与工程人才断代。
- 战争后知识被炸毁、封存、污染、垄断。
- 中阶认证法师稀缺，边境排不上队。
- 高阶以上人才不进入公共服务体系。
- 缺少标准化测试。
- 缺少材料数据库。
- 缺少工艺记录。
- 缺少批量培训体系。
- 缺少低成本替代方案。
- 无法重新定义低阶魔法能力。

# 09 王国权力结构 canon

本书所在王国不是现代中央集权国家，而是后高魔时代的分权王国。

基本结构：

```text
王室财政派
+ 北境军方 / 军需署
+ 半独立法师行会
+ 大贵族封地集团
+ 商会渠道集团
+ 边境小领主
+ 邻国压力
```

国家存在，但不是统一意志。

主角不能独立对抗国家机器。他的中期生存逻辑是：

```text
从灰石领小男爵转化为北境军需体系的战略供应商，
以边境军方订单和王室财政派背书，
抵抗法师行会认证封锁、商会买断和大贵族吞并。
```

这种保护不是免费的。

军方和王室会提供采购、特许、护卫和合法性，同时要求优先供货、技术备案、质量审查、军用型号控制和政治站队。

主角的中后期冲突，不是自由发展，而是在强力保护伞下争取工坊自治、贡献契约和低阶技师体系的独立性。

# 10 强机构合作 canon

## 北境军需署

北境军需署是主角中期最重要的第一保护伞。

它需要灰炉体系解决：

- 冬季驻军冻伤。
- 哨塔供暖。
- 军粮仓储。
- 矿区排水。
- 运输驿站。
- 高价行会维修依赖。
- 劣质魔石利用。

北境军需署给主角：

- 军需采购订单。
- 低魔耗供暖设备试制资格。
- 抵抗地方行会封禁的文件。
- 有限军方保护。
- 进入北境军需网络的渠道。

北境军需署要求主角：

- 优先供应军方。
- 接受质量审查。
- 提供事故记录。
- 培训军方维修员。
- 军用型号不得外售。

## 王室财政派

王室财政派是主角从区域工坊走向王国合法性的第二保护伞。

它需要灰石体系：

- 降低旧魔导维护成本。
- 制衡法师行会认证垄断。
- 恢复边境税源。
- 降低北境军需开支。
- 增加王室对边境的控制工具。

王室财政派给主角：

- 王室特许状。
- 税务减免。
- 北境制造和维修许可。
- 对抗行会封锁的合法性。
- 王室采购资格。

王室财政派要求主角：

- 技术备案。
- 王室监督。
- 税收上缴。
- 军用设备优先权。
- 不得向敌国出售关键型号。

# 11 灰石领地理与商业半径 canon

灰石领起点不是商业中心。

它是高魔时代后衰败的边境旧节点，位于寒岭南麓，卡在以下位置之间：

- 旧军道。
- 低品位矿区。
- 边境哨塔补给线。
- 废弃魔晶矿支线。
- 旧冬路商道。
- 寒地村镇带。

高魔时代，灰石领曾依赖魔晶支线、军道转运和边境补给维持价值。烬日战争后，魔力基础设施崩坏，调律塔失效，魔晶矿品位下降，旧军道荒废，灰石领因此衰败。

陆衡不是在原地把一个小领地凭空做成世界中心，而是：

```text
先用灰炉救活本地，
再恢复废弃矿区、旧军道驿站和边境哨塔补给，
再建立灰炉驿站、授权维修点、低阶技师培训点和军需仓储节点，
最终形成北境低魔工业走廊。
```

## 灰炉走廊

中后期主角真正控制的不是单座城市，而是一条低魔工业供应走廊：

```text
灰石城
→ 灰炉驿站
→ 军需仓库
→ 矿区排水站
→ 授权维修点
→ 控温员培训所
→ 商队供暖站
→ 边境哨塔
```

灰石领的成长不是城市凭空繁荣，而是从一个衰败节点重新变成北境供暖、军需、矿区、维修、技师培训和冬路商贸的枢纽。

# 12 主角工业路线 canon

陆衡穿越后不直接从蒸汽机开始。

蒸汽动力需要可靠锅炉、耐压材料、密封件、阀门、管道、压力表、安全阀、轴承、加工精度和维修体系。灰石领开局没有这些条件。

第一卷的正确工程起点是生存热工：

```text
灰炉一型
→ 耐火材料
→ 控温岗位
→ 劣质魔石波动测试
→ 炉温记录
→ 低魔耗燃烧
→ 工坊贡献契约雏形
```

长篇工业演进路线：

```text
生存热工
→ 工坊标准化
→ 金工与机械基础
→ 低压动力
→ 交通物流
→ 城镇基础设施
→ 铭文仪表与控制
→ 低魔力网络
```

这条路线不是复原旧高魔魔网，而是建立低压、低耗、分布式、可维护、可追责的新魔法工业网络。

# 13 魔法能力工业化原则

主角不会把魔法当成神秘天赋，而是拆成可记录、可训练、可替换、可协作的工序能力。

核心价值观：

```text
战斗看上限，工业看稳定。
天赋看血统，工序看可重复。
秘方看垄断，工程看记录。
魔法能力不是职业身份，而是工序能力。
```

六系岗位化：

```text
火系：热工岗位，控温、热处理、烧结、烘干、锅炉点火。
土系：土石 / 陶工 / 基建岗位，成型、夯实、矿石粗选、模具、地基。
金系：金工 / 金属材料岗位，去杂、锻造辅助、管件、轴承、裂纹检测、标准件。
风系：流体 / 气动岗位，鼓风、通风、风选、减阻、交通助推。
水系：循环 / 冷却 / 清洗岗位，冷却循环、水泵、洗矿、冷凝、排水。
铭文系：仪表 / 控制 / 认证岗位，读数、报警、联锁、记录、防篡改、认证。
```

第一卷主线只重点使用火系、土系、铭文系。金系、风系、水系可作为伏笔或辅助，不抢灰炉主线。

# 14 成果保护 canon

主角保护成果不能停留在契约和署名。

第一卷中，契约只能保护源头和底线：

- 实验记录。
- 参与者署名。
- 巴伦见证。
- 公开测试事实。
- 工坊贡献契约。

中期保护必须依靠强机构和产能：

- 北境军需署采购。
- 王室特许。
- 标准件供应。
- 技师培训。
- 维修网络。
- 授权工坊。
- 灰炉铭牌。

后期保护必须依靠战略依赖和工业武力：

- 灰炉走廊。
- 灰石城。
- 工业防御体系。
- 小领主采购网络。
- 北境军需依赖。
- 王室财政收益。
- 行会实用派合作。
- 低魔网络多方制衡。

主角真正保护的不是一张图纸，而是让知识、工序、人员、署名、维护、责任和改进能力形成别人无法低成本吞并的体系。

# 15 爵位与权力成长 canon

全书主线十二年。

```text
灰石男爵
→ 王室特许工业男爵
→ 灰石子爵
→ 北境工业伯
→ 寒岭侯
```

## 灰石男爵

第一卷起点。

代表最低限度法理：查账、开工坊、调领民、签延期、拒绝买断。

## 王室特许工业男爵

爵位不变，但叠加军需试制资格和王室有限特许。

代表主角进入强机构视野。

## 灰石子爵

触发条件：恢复废弃矿区、旧军道驿站和周边破产村镇，稳定北境局部军需供暖。

代表灰石领从本地求生进入区域供应。

## 北境工业伯

触发条件：灰石城初步形成，灰炉走廊稳定，低阶技师学校成型，行会认证战争第一阶段胜利。

代表灰石体系成为北境工业贵族势力。

## 寒岭侯

触发条件：寒线战争中，灰石工业体系保障北境军需、守住寒岭防线、证明低魔工业体系具备国家级战略价值。

寒岭侯是终点。

不建议走到公爵、亲王、建国或王位继承线。

# 16 关键配角 canon

本文件只保留既有已定角色 canon，不新增女主或强机构代表角色。

## 巴伦

- 身份：格雷家族世袭管家兼旧家臣。
- 核心欲望：守住格雷家最后的尊严。
- 初始态度：怀疑主角变化，反对主角进入工坊。
- 变化方向：从守旧贵族体面，到见证新工坊契约。
- 写作重点：巴伦不能无脑信任主角。他接受的不是“陆衡”，而是“这个变化后的莱恩至少正在承担格雷家过去没做到的事”。

## 埃文·巴伦

- 身份：巴伦的孙子，火系魔法学徒。
- 性别：男性。
- 合理性：家臣家庭有资格接受魔法启蒙。
- 传统评价：火焰弱、爆发低、无战斗价值，被行会退回。
- 主角评价：火焰弱但稳定，适合作为灰炉控温员。
- 弧光：失败学徒 → 一号炉温控制员 → 第一代魔法工业技师。
- 写作重点：埃文的爽点不是弱火突然变强，而是弱火被重新定义为稳定控温价值。

## 维克多

- 身份：法师行会中阶认证法师。
- 立场：魔导器危险，必须认证，否则可能害人。
- 功能：旧认证秩序代表，不是小丑反派。
- 写作重点：维克多的合理性来自安全、规程和事故责任。他与主角争的是行会认证体系 vs 工程验证体系。
- 战力边界：维克多有认证和审查压迫力，但不是能随手摧毁灰石领的高阶战斗法师。

## 赫曼

- 身份：边境商会代表。
- 立场：成果可以被合同买断，技术可以变成垄断商品。
- 功能：前世伤口镜像。
- 写作重点：赫曼不是低级奸商。他的合同必须合法、诱人、危险，能提供灰石领急需现金，同时要求图纸、制造权和署名控制。

# 17 第一卷目标

第一卷《灰炉与寒潮》只解决一件事：灰石领如何活过寒潮。

技术线：灰炉一型从失败到可用。

人物线：陆衡从否认身份，到被迫使用莱恩身份，再到第一次主动用这个身份保护他人劳动成果。

制度线：从个人试错到工坊贡献契约。

外部压力：寒潮、欠税、商会买断、行会调查。

第一卷不写帝国大战，不写高级魔法理论，不写末法历史真相，不让北境军需署、王室财政派、大贵族或邻国正式下场。

第一卷只呈现末法现象：旧暖炉停摆、魔石变差、维修昂贵、行会认证慢、领民用不起旧设备、灰石领撑不过寒潮。

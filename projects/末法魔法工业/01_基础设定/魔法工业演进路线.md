## 基本信息

- 项目编号：末法魔法工业
- 质感长篇名：《灰炉纪元》
- 番茄男频爽文名：《末法时代，我把魔法做成工业》
- 当前阶段：基础设定 / 长篇工业路线确立
- 当前文件版本：v0.2
- 文件定位：长篇产业升级母文件

## 文件用途

本文件用于统一整部长篇的魔法工业演进逻辑，回答三个底层问题：

1. 高魔时代为什么突然进入末法时代。
2. 高魔工业明明强大，为什么会出现能源、材料、人才、知识、制度与思维断层。
3. 陆衡穿越后为什么不能直接从蒸汽机开始，而必须从灰炉一型、生存热工和低魔耗工序体系起步。

后续设计卷纲、技术产品、六系岗位化、行会冲突、商会买断和新工业制度时，必须读取本文件。

# 01 高魔时代与末法断层

## 高魔时代的真正基础

高魔时代的魔法工业强大，不是因为单个法师都很强，而是因为整个大陆曾经拥有一套庞大的魔力基础设施。

高魔工业的基础结构：

```text
地脉魔力
→ 魔晶矿脉
→ 魔晶炉群
→ 地脉锚点
→ 调律塔
→ 城市魔网
→ 行会认证接入
→ 贵族、工坊、城市公共设备使用
```

高魔时代的辉煌产品包括：

- 魔晶炉
- 魔导水泵
- 贵族供暖系统
- 浮空运输
- 自动铭文机
- 城市魔网
- 行会认证工坊
- 贵族家族秘方
- 大型魔导战争器械

## 断层事件：烬日战争

距今约一百年前，各大王国、贵族集团和法师势力围绕地脉节点、魔晶矿脉和城市魔网展开大战。

战争后期，双方开始攻击彼此的魔力基础设施：

- 地脉锚点
- 调律塔
- 魔晶炉群
- 城市魔网中枢
- 高阶铭文校准站
- 军用魔导仓库

最终，禁忌级战略魔导器被启动，造成主大陆魔力潮汐失衡。

世界并未失去魔法，但失去了高魔工业所需的稳定、高纯、高浓度魔力输入。

## 末法时代的本质

末法不是魔法完全消失，而是输入条件改变：

```text
过去：高浓度、高纯度、高稳定性的魔力输入。
现在：低浓度、高波动、高杂质的魔力输入。
```

因此旧魔导工业不是一夜消失，而是进入全面失配：

- 魔力浓度下降。
- 魔力频率变乱。
- 魔石杂质化。
- 地脉节点断续。
- 高阶法术施放成本暴涨。
- 旧魔导设备在劣质魔石下频繁熄火、倒灌、过热、裂损。
- 边境小领地无法承担旧设备维护成本。

# 02 断层时间表

采用约九十六年的主断层周期。

```text
距今 120 年：高魔时代末期，魔导工业达到巅峰。
距今 104 年：烬日战争爆发，各国争夺地脉节点与魔晶矿脉。
距今 96 年：禁忌级战略魔导器启动，地脉锚点崩坏，主大陆进入末法。
距今 90-70 年：灾后抢修期，各国依靠库存魔石、旧设备和幸存大师硬撑。
距今 70-40 年：行会认证权强化，贵族秘方封闭，商会掌握魔石与旧零件供应链。
距今 40-10 年：边境领地陆续衰败，平民被重新挤出魔导消费体系。
距今 0 年：灰石领旧暖炉停摆，寒潮将至，陆衡穿越为莱恩·格雷。
```

三代人结构：

1. 第一代亲历高魔战争，死伤惨重。
2. 第二代靠旧设备和库存硬撑，建立封闭认证制度。
3. 第三代出生在末法后，只知道旧设备越来越贵、越来越难修。

这个时间尺度可以保留高魔遗产，又足以造成知识、人才和制度断层。

# 03 为什么没有自然研究出末法工业

## 一、能源断层

旧魔导设备依赖稳定高魔输入。末法后，世界剩下低浓度、高波动、高杂质魔力。

这不是简单的“能量少”，而是类似整个能源网络的电压、频率、纯净度和稳定性全部失配。

旧设备还能运行，但故障率和维护成本暴涨。

## 二、材料断层

高魔时代依赖高品质魔石、高魔浸润材料和旧合金。

末法后：

- 魔石杂质增加。
- 矿脉品位下降。
- 旧合金失去魔力浸润后变脆。
- 铭文载体老化。
- 耐火材料在不稳定魔力冲击下更容易裂损。

这给陆衡的材料学能力留下切入点。

## 三、人才断层

战争杀死的不只是战斗法师，还包括地脉调律师、魔晶炉工程师、城市魔网维护师、高阶铭文校准师和魔导材料大师。

战后还存在会使用设备、维护表层和背诵规程的人，但真正懂底层重建的人已经断代。

法师行会从研发组织逐渐退化为维护与认证组织。

## 四、知识断层

高魔时代核心知识在战争后出现四种状态：

```text
被炸毁
被封存
被污染
被垄断
```

知识被切碎：

- 贵族家族掌握部分秘方。
- 法师行会掌握部分认证标准。
- 商会掌握部分供应链记录。
- 旧军方掌握部分禁忌技术。
- 工匠掌握经验，但缺少公开记录体系。
- 平民几乎没有进入知识体系的资格。

## 五、制度断层

资源减少后，掌握旧设备、旧秘方和认证权的人更不愿放权。

法师行会强调安全认证，贵族强调家族秘方，商会强调渠道和买断。三者都有合理性，但合在一起，阻止了低成本、可复制、可培训的新工业自然诞生。

## 六、思维断层

旧世界的研发方向仍然是：

```text
更强的火焰
更纯的魔石
更高级的铭文
更复杂的法阵
更高阶的法师
```

末法时代真正需要的是：

```text
低魔耗
低成本
稳定
可维护
可复制
可培训
可署名
可追责
```

陆衡的时代优势，不是知道答案，而是来自资源受限条件下的工程优化思维。

# 04 主角穿越后的工业现状

陆衡穿越时，这个世界不是正常中世纪，也不是标准蒸汽前夜。

它的真实状态是：

```text
后高魔遗产社会
+ 手工业生产
+ 行会认证维护体系
+ 边境工业退化
```

上层世界仍保有少量昂贵旧魔导设备：

- 城市供暖
- 魔导水泵
- 贵族温室
- 浮空运输残线
- 高价通讯塔
- 行会认证工坊

但边境小领地已经退化到：

- 手工锻造
- 人力搬运
- 木炭和劣质魔石混用
- 旧设备靠拆零件续命
- 工坊缺少标准件
- 账本混乱
- 领民取暖困难
- 行会维修昂贵且缓慢

灰石领旧暖炉停摆，不是因为魔法不存在，而是因为旧设备无法适应劣质魔石和低魔环境。

# 05 为什么不直接从蒸汽机开始

陆衡穿越后不应直接制造蒸汽机。

真正可用的蒸汽动力需要前置条件：

- 可靠锅炉
- 耐压材料
- 密封件
- 阀门
- 管道
- 压力表
- 安全阀
- 轴承
- 加工精度
- 冷凝或排汽处理
- 维修体系

灰石领开局没有这些条件。

第一卷的现实问题不是动力，而是生存：

```text
寒潮来了。
旧暖炉停了。
领民要冻死。
税务官三天后到。
领地没有钱。
```

因此最合理的工程选择是先做灰炉一型：低魔耗蓄热炉。

灰炉一型不是低级产品，而是后续动力系统的前置技术。它解决：

- 稳定热源
- 耐火材料
- 控温岗位
- 炉体结构
- 劣质魔石利用
- 热效率
- 安全测试
- 工序记录

# 06 工业演进路线

## 第一阶段：生存热工

代表产品：灰炉一型。

核心问题：活过寒潮。

主要内容：低魔耗蓄热炉、耐火砖、炉温记录、劣质魔石波动测试、火系控温岗位、工坊贡献契约雏形。

主导系别：火系、土系、铭文系。

## 第二阶段：工坊标准化

代表产品：标准炉芯、耐火砖批次、工序卡。

核心问题：让成功可复制。

主要内容：炉芯模块、标准耐火砖、工序记录、学徒培训、简易验收、工坊贡献契约。

主导系别：火系、土系、铭文系，金系开始作为工具和标准件问题出现。

## 第三阶段：金工与机械基础

代表产品：炉门、管件、阀门、轴承、标准件。

核心问题：解决扩产和机械可靠性。

主要内容：金属裂纹检测、管件修补、轴承打磨、标准螺栓 / 铆钉、手摇鼓风机、小型金属加工工序。

主导系别：金系、火系、水系、土系、铭文系。

## 第四阶段：低压动力

代表产品：水泵、鼓风机、升降机、小型动力轴。

核心问题：让工坊从人力走向机械辅助。

主要内容：低压蒸汽辅助设备、矿井排水、工坊动力台、过压报警、安全阀和阀门事故。

主导系别：火系、风系、水系、金系、铭文系。

## 第五阶段：交通物流

代表产品：风助货车、矿道牵引、灰炉动力拖车。

核心问题：产品运出去，资源运进来。

分工规则：

```text
风系：推力、减阻、侧风补偿。
金系：车轴、轮毂、轴承、轨道、结构强度。
土系：道路、路基、桥梁、轨道基础。
火系：灰炉动力、防冻、热源。
水系：冷却、循环、湿滑处理。
铭文系：限速、报警、稳定、制动联锁。
```

风系可以给交通工具提速，但不等于交通工具本身。

## 第六阶段：城镇基础设施

代表产品：供暖、水网、排污、消防、粮仓通风。

核心问题：工业带来人口聚集和城镇管理。

主要内容：集中供暖、热水管线、水泵站、排水沟、粮仓通风、工坊消防、城镇道路硬化。

主导系别：水系、土系、火系、风系、金系、铭文系。

## 第七阶段：铭文仪表与控制

代表产品：压力表、温标、报警铭文、安全联锁、记录防篡改。

核心问题：设备不能只靠人盯，必须可读、可控、可追责。

主要内容：温度表、压力表、批次铭文、过载切断、事故追踪铭牌、新式公开测试标准。

主导系别：铭文系 + 全系协作。

## 第八阶段：低魔力网络

代表产品：工坊动力分配、低耗通讯、区域供能节点。

核心问题：重建适合末法时代的新魔法工业网络。

这不是复原高魔魔网，而是建立低压、低耗、分布式、可维护的新工业网络。

# 07 六系岗位化总原则

主角不会把魔法能力看作职业身份，而是拆成可记录、可训练、可替换、可协作的工序能力。

核心原则：

```text
战斗看上限，工业看稳定。
天赋看血统，工序看可重复。
秘方看垄断，工程看记录。
魔法能力不是职业身份，而是工序能力。
```

六系岗位化：

| 系别 | 工业定位 | 主要加入阶段 |
|---|---|---|
| 火系 | 热工岗位：控温、热处理、锅炉点火、烧结、退火 | 第一阶段即主线 |
| 土系 | 土石 / 陶工 / 基建岗位：成型、夯实、模具、矿石粗选、地基 | 第一阶段即主线 |
| 金系 | 金工 / 金属材料岗位：去杂、锻造辅助、管件、轴承、裂纹检测、标准件 | 第三阶段正式主线 |
| 风系 | 流体 / 气动岗位：鼓风、通风、风选、减阻、交通助推 | 第四、第五阶段主线 |
| 水系 | 循环 / 冷却 / 清洗岗位：冷却循环、水泵、洗矿、冷凝、排水 | 第四、第六阶段主线 |
| 铭文系 | 仪表 / 控制 / 认证岗位：读数、报警、联锁、记录、防篡改、认证 | 第一阶段出现，第七阶段爆发 |

# 08 第一卷信息释放限制

第一卷只允许呈现末法现象，不揭示完整历史真相。

允许释放：

- 旧暖炉停摆。
- 魔石质量下降。
- 行会维修昂贵。

暂不揭示：

- 烬日战争全貌。
- 地脉锚点崩坏细节。
- 调律塔和城市魔网结构。
- 禁忌战略魔导器真相。
- 高魔时代产业体系全貌。
- 低魔力网络的未来蓝图。

第一卷所有世界信息必须通过场景压力出现，而不是百科说明。

# 01 总原则

## 一句话工业路线

陆衡不是把现代科技直接搬进异界，而是在一个高魔基础设施崩坏、旧魔导工业失配、魔法师阶层仍强的末法世界里，用现代工程方法把低阶魔法能力拆成岗位，把失败变成记录，把设备变成标准，把标准变成产能，把产能变成军需与民生依赖，最终形成低压、低耗、分布式、可维护、可追责的低魔工业体系。

## 路线核心

```text
不是恢复高魔时代。
不是直接造蒸汽机。
不是靠个人战力打败高阶法师。
不是靠一纸契约保护技术。

而是：
从生存热工开始，
逐步建立工坊标准、金工机械、低压动力、交通物流、城镇基础设施、铭文控制和低魔网络，
并让工业体系同步生长出财政、武力、政治背书、市场半径和战略依赖。
```

## 主角竞争维度

旧法师竞争：

```text
个人上限、血统、等级、爆发、法阵复杂度、高品质魔石和贵族资源。
```

陆衡竞争：

```text
单位成本、稳定产出、复现速度、维护周期、培训门槛、事故追责、制度吸引力和战略后勤价值。
```

高阶法师可以打穿一个点，但不能低成本支撑一个寒地社会的持续供暖、维修、矿区排水、军需仓储、运输驿站和低阶技师培训网络。

# 02 高魔时代到末法时代

## 高魔工业基础结构

高魔时代的魔法工业不是靠单个法师强，而是靠基础设施强。

```text
地脉魔力
→ 魔晶矿脉
→ 魔晶炉群
→ 地脉锚点
→ 调律塔
→ 城市魔网
→ 行会认证接入
→ 贵族、工坊、城市公共设备使用
```

旧魔导器默认可以获得稳定、高纯、高浓度魔力输入。

## 烬日战争与输入失配

距今约九十六年前，烬日战争末期，各方攻击彼此的地脉锚点、调律塔、魔晶炉群、城市魔网中枢和高阶铭文校准站，并动用禁忌级战略魔导器，造成主大陆魔力潮汐失衡。

世界并未失去魔法，但失去了高魔工业所需的稳定输入：

```text
过去：高浓度、高纯度、高稳定性的魔力输入。
现在：低浓度、高波动、高杂质的魔力输入。
```

## 末法后的真实状态

```text
后高魔遗产社会
+ 手工业生产
+ 行会认证维护体系
+ 边境工业退化
+ 贵族与军方争夺残余魔导资源
```

上层仍保有昂贵旧魔导设备。

边境小领地退化到手工锻造、人力搬运、木炭和劣质魔石混用、旧设备靠拆零件续命、行会维修昂贵且缓慢的状态。

# 03 六重断层

| 断层 | 内容 | 对主角路线的意义 |
|---|---|---|
| 能源断层 | 稳定高魔输入变为低浓度、高波动、高杂质输入 | 灰炉必须低魔耗、抗波动、可缓冲 |
| 材料断层 | 魔石杂质增加，旧合金和铭文载体老化，耐火材料更易裂损 | 陆衡材料学能力有切入点 |
| 人才断层 | 地脉调律师、魔晶炉工程师、城市魔网维护师、高阶铭文校准师断代 | 行会从研发组织退化为维护认证组织 |
| 知识断层 | 高魔知识被炸毁、封存、污染、垄断 | 公开记录、失败样本和工序卡具有革命意义 |
| 制度断层 | 行会、贵族、商会都守旧设备、旧秘方和旧供应链 | 新工业必须获得军需和王室背书才能活下去 |
| 思维断层 | 旧体系追求更强、更纯、更高阶 | 主角转向低魔耗、低成本、稳定、可维护、可培训 |

# 04 为什么不能从蒸汽机开始

陆衡穿越后不直接从蒸汽机开始。

蒸汽动力需要：

- 可靠锅炉。
- 耐压材料。
- 密封件。
- 阀门。
- 管道。
- 压力表。
- 安全阀。
- 轴承。
- 加工精度。
- 维修体系。
- 操作员培训。
- 事故责任制度。

灰石领开局没有这些条件。

第一卷的正确工程起点是生存热工：

```text
灰炉一型
→ 耐火材料
→ 控温岗位
→ 劣质魔石波动测试
→ 炉温记录
→ 低魔耗燃烧
→ 工坊贡献契约雏形
```

灰炉一型不是终极技术，而是低魔工业的第一块炉芯。

# 05 长篇工业演进八阶段

```text
生存热工
→ 工坊标准化
→ 金工与机械基础
→ 低压动力
→ 交通物流
→ 城镇基础设施
→ 铭文仪表与控制
→ 低魔力网络
```

这条路线不是复原旧高魔魔网，而是建立低压、低耗、分布式、可维护、可追责的新魔法工业网络。

每一阶段都必须同时回答：

```text
技术上解决什么？
市场从哪里来？
谁会保护它？
谁会抢夺它？
它需要什么武力和制度支撑？
它会给主角带来什么新的束缚？
```

# 06 阶段一：生存热工

## 时间位置

第 0 年 / 第一卷《灰炉与寒潮》。

## 核心产品

灰炉一型，低魔耗蓄热炉。

## 技术目标

- 使用劣质魔石。
- 缓冲魔力波动。
- 依赖低阶火系稳定控温，而非高阶火系爆发。
- 使用灰石领本地粘土、灰石、旧炉残件。
- 建立最早的炉温记录和失败记录。

## 六系使用

- 火系：埃文弱火控温。
- 土系：耐火砖、炉体成型、材料处理。
- 铭文系：旧炉封印、读数、基础报警、认证压力。
- 金系、风系、水系只作伏笔。

## 政治与市场

- 市场半径：灰石领本地。
- 政治背书：只有男爵法理和巴伦家臣秩序。
- 武力保护：几乎没有，只能靠边境太小、不被重视、工坊内部秩序。
- 敌人压力：税务官、地方商会、地方法师行会。

## 阶段成果

灰石领活过寒潮，灰炉一型勉强可用，工坊贡献契约出现。

## 阶段禁区

- 不进入蒸汽机。
- 不量产。
- 不让军需署和王室正式背书。
- 不解释末法历史真相。

# 07 阶段二：工坊标准化

## 时间位置

第 1-3 年。

## 核心任务

把一台灰炉，变成可小批量复制、可安装、可维修、可追责的工坊产品。

## 技术内容

- 炉芯砖配方稳定。
- 粘土含水率记录。
- 烧制曲线。
- 魔石分级。
- 控温员培训。
- 安装步骤。
- 维修手册。
- 灰炉铭牌。
- 事故记录。
- 基础防篡改铭文。

## 六系使用

- 火系：控温员岗位成型。
- 土系：耐火砖批次稳定。
- 铭文系：铭牌、记录、防篡改、温标。
- 金系：炉门、铰链、金属箍疲劳问题开始暴露。
- 风系：通风、烟道、鼓风不足。
- 水系：冷却、清洗、热冲击实验。

## 政治与市场

- 市场半径：周边村镇、边境哨塔。
- 政治背书：北境军需署开始试用。
- 武力保护：工坊护卫、仓库防盗、记录副本。
- 敌人压力：盗版灰炉、商会挖人、行会认证警告。

## 阶段成果

灰石领成为北境低魔耗供暖设备试制工坊。

## 阶段代价

军需署带来保护，也带来质量审查、军用型号限制、监督员介入。

# 08 阶段三：金工与机械基础

## 时间位置

第 4-6 年，灰石子爵阶段。

## 核心任务

解决灰炉扩散后暴露出的机械基础瓶颈。

灰炉做大后，瓶颈从炉子本体转向：

```text
金属件
管件
阀门
铰链
轴承
标准工具
矿区排水
维修零件
```

## 技术内容

- 金属裂纹检测。
- 金属疲劳识别。
- 管件和阀门制造。
- 轴承打磨。
- 标准件编号。
- 矿石品位识别。
- 矿区排水设备。
- 水泵初型。
- 工具维修体系。

## 六系使用

- 金系正式进入主线：去杂、裂纹检测、锻造辅助、标准件。
- 水系进入矿区排水、洗矿和冷却循环。
- 火系继续承担热处理。
- 土系承担矿道加固和工坊地基。
- 铭文系承担批次、铭牌、维修记录。

## 政治与市场

- 市场半径：灰石领 + 废弃矿区 + 旧军道驿站 + 周边小领地。
- 政治背书：王室有限特许、北境军需署采购。
- 武力保护：矿区护卫、工坊巡逻、运输队。
- 敌人压力：贵族夺矿、商会控制标准件渠道、行会加强认证压力。

## 阶段成果

灰石领恢复废弃矿区和旧军道驿站，晋升灰石子爵。

## 阶段代价

产能越重要，越容易被强权盯上。主角必须开始把工业收益转化为护卫、仓储、防御和政治交易。

# 09 阶段四：低压动力

## 时间位置

第 5-7 年，与金工、交通物流部分重叠。

## 核心任务

从纯供暖设备进入小型机械辅助，不追求大型蒸汽工业，而是先解决边境最急迫的低压动力需求。

## 技术内容

- 低压水泵。
- 鼓风机。
- 简易升降机。
- 小型动力轴。
- 矿区排水动力。
- 工坊风箱改良。
- 安全阀雏形。
- 压力报警铭文。

## 六系使用

- 火系：稳定热源。
- 水系：循环、泵送、冷却。
- 风系：鼓风和气流控制。
- 金系：管件、阀门、轴承。
- 铭文系：压力阈值、报警、联锁。

## 政治与市场

- 市场半径：矿区、军需仓库、工坊区。
- 政治背书：北境军需署和王室财政派都开始关注。
- 武力保护：矿区护卫、设备仓库、维修队。
- 敌人压力：军方要求优先供应，行会要求纳入认证。

## 阶段成果

灰石体系不再只是供暖，而开始支撑矿区、工坊和军需后勤。

## 阶段禁区

低压动力不是完整蒸汽时代，不写高速铁路、大型蒸汽舰、现代工厂流水线。

---

# 10 阶段五：交通物流

## 时间位置

第 5-7 年，灰炉走廊形成期。

## 核心任务

解决“东西造出来后，能不能运出去、修得起、维持得住”。

## 技术内容

- 风助货车。
- 冬季商道供暖站。
- 矿道牵引。
- 商队维修车。
- 灰炉驿站。
- 低温仓储。
- 道路夯实和桥梁维修。
- 运输线铭文标记。

## 六系使用

- 风系：助推、减阻、通风、风选。
- 金系：车轴、轮毂、连接件、维修工具。
- 土系：道路、桥基、驿站地基。
- 火系：驿站供暖、维修热源。
- 水系：路面清洗、冻结控制、冷却。
- 铭文系：路线标记、限速、报警、货物防篡改。

## 政治与市场

- 市场半径：旧军道、冬路商道、边境哨塔、周边领地。
- 政治背书：王室特许经营权。
- 武力保护：护商队、山口哨站、运输护卫。
- 敌人压力：邻国间谍、盗匪、商会渠道封锁、贵族路权争夺。

## 阶段成果

灰炉走廊形成，灰石领从本地工业点转化为北境低魔设备供应网络核心。

# 11 阶段六：城镇基础设施

## 时间位置

第 7-9 年，北境工业伯阶段。

## 核心任务

灰石领从工坊群升级为工业城镇。

## 技术内容

- 集中供暖站。
- 工坊区规划。
- 粮仓通风。
- 水循环。
- 排水沟。
- 消防池。
- 工人宿舍。
- 技师学校。
- 标准件市场。
- 授权维修站网络。

## 六系使用

- 火系：集中供暖和工坊热源。
- 水系：水网、消防、排污、清洗。
- 土系：道路、地基、仓库、墙体。
- 风系：通风、粮仓防霉、烟道。
- 金系：管网、阀门、工具、标准件。
- 铭文系：温标、压力、报警、记录和防篡改。

## 政治与市场

- 市场半径：灰石城 + 灰炉走廊核心段 + 北境设备网络。
- 政治背书：王室与军方共同承认，晋升北境工业伯。
- 武力保护：城防雏形、工坊护卫营、运输护商队、仓库防御。
- 敌人压力：王都收编、行会封锁、大贵族监管、商会利益争夺。

## 阶段副作用

工业化必须带来副作用：

- 工人伤亡。
- 学徒过劳。
- 工坊事故。
- 技术泄露。
- 贫民涌入。
- 粮食压力。
- 治安问题。
- 旧贵族礼法崩解。
- 低阶技师和传统法师冲突。

主角必须面对：新体系会不会变成另一种压迫。

# 12 阶段七：铭文仪表与控制

## 时间位置

第 7-10 年，与城镇基础设施和行会认证战争重叠。

## 核心任务

让设备不再只靠人盯，而是进入读数、报警、联锁、记录、防篡改和认证体系。

## 技术内容

- 温标。
- 压力表。
- 报警铭文。
- 安全切断。
- 炉芯批次追踪。
- 设备铭牌。
- 事故记录。
- 控温员考核。
- 维修资格。
- 授权工坊备案。

## 政治含义

这一阶段会正面挑战法师行会。

行会旧认证依赖：

```text
等级、师承、封闭经验、行会许可。
```

灰石新认证依赖：

```text
公开测试、批次记录、事故追责、操作资格、维修标准、防篡改铭牌。
```

## 政治与市场

- 市场半径：北境低魔设备网络。
- 政治背书：王室财政派、北境军需署、行会实用派部分参与。
- 武力保护：城防、工坊护卫、记录库保护。
- 敌人压力：行会保守派、中央派系、贵族派。

## 阶段成果

成立半官方审查机构，例如：

```text
北境工业魔导试验署
```

它由军方、王室、行会实用派和灰石领共同参与，不完全属于任何一方。

# 13 阶段八：低魔力网络

## 时间位置

第 11-12 年及终局。

## 核心任务

建立低压、低耗、分布式、可维护、可追责的新魔法工业网络。

## 不是复原高魔魔网

低魔力网络不是高魔时代城市魔网的复原。

它必须具有：

```text
低压
低耗
分布式
可维护
可追责
可切断
可分区
多方制衡
```

## 用途

- 工坊供能。
- 温度监控。
- 压力报警。
- 仓储状态。
- 低耗通信。
- 城镇供暖调度。
- 军需节点协调。
- 灰炉走廊状态监控。

## 最终争夺

所有强权都想控制低魔网：

- 王室想国家化。
- 行会想认证化。
- 军方想军管化。
- 商会想收费化。
- 贵族想封地私有化。
- 主角想分布式、可追责和多方制衡。

## 阶段成果

低魔网络标准确立，灰石体系从设备供应商转化为北境低魔工业基础设施核心。

## 主题风险

如果低魔网络被单方垄断，它会变成第二个高魔时代贵族魔网。

主角终局必须推动多方制衡，而不是把全部控制权据为己有。

# 15 成果保护随工业升级

## 第一阶段：源头保护

第一卷只能做到：

- 实验记录。
- 参与者署名。
- 巴伦见证。
- 公开测试事实。
- 工坊贡献契约。

## 第二阶段：工序保护

通过工序卡、批次记录、安装标准、控温员训练和维修手册，让别人偷图纸也复制不了稳定产出。

## 第三阶段：产能保护

通过耐火砖、标准件、管件、阀门、水泵、维修件和工具体系，让灰石领掌握关键耗材和维护能力。

## 第四阶段：强机构保护

通过北境军需署采购和王室特许，让商会、地方行会和小贵族无法无成本吞并。

## 第五阶段：武力保护

通过工坊护卫、矿区护卫、护商队、城防体系和工业防御体系保护产线、仓库、技师和记录库。

## 第六阶段：战略依赖保护

当北境供暖、军需仓储、矿区排水、冬路驿站、标准件维修和低阶技师培训都依赖灰石体系时，强行吞并的成本会高到所有派系都必须计算后果。

最终原则：

```text
主角保护的不是一张图纸，
而是一套别人无法低成本吞并的产能、人员、标准、维修、军需和利益共同体。
```

# 16 第一卷工业边界

第一卷只进入生存热工。

允许：

- 旧炉拆解。
- 劣质魔石波动测试。
- 耐火砖失败。
- 含水率记录。
- 埃文弱火控温。
- 炉温曲线。
- 灰炉一型勉强可用。
- 公开测试。
- 工坊贡献契约。

禁止：

- 直接进入蒸汽机。
- 直接建立标准件体系。
- 大规模量产。
- 金系正式成为主线。
- 风系交通正式展开。
- 水系城建正式展开。
- 北境军需署正式背书。
- 王室财政派正式介入。
- 低魔网络蓝图公开。

# 17 长篇工业禁区

- 不让主角靠现代知识一步到位。
- 不让本土旧体系全员愚蠢。
- 不让魔法师阶层突然变弱。
- 不让工业扩张没有原料、工具、标准件和维修体系。
- 不让产品扩散没有运输、市场、军需和政治背书。
- 不让成果保护停留在契约和署名。
- 不让灰石领凭空成为商业中心。
- 不让爵位、领地、市场半径和工业影响力脱节。
- 不让低魔网络成为主角个人私有魔网。
- 不让最终路线滑向公爵、亲王、建国或王位继承线。

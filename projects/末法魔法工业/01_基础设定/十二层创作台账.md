# 01 核心命题

真正改变世界的不是某个天才个人，而是能让知识被验证、被传承、被署名、被保护，并让普通人参与创造的体系。

# 02 读者承诺

## 爽感承诺

- 现代工程方法拆解旧魔法工业。
- 低阶魔法能力被重新定义为工业岗位。
- 劣质资源被重新利用。
- 破落领地从第一台灰炉开始翻身。
- 旧体系判定无价值的人和物，被主角用记录、测试和工序重新定义。

## 成长承诺

- 陆衡从不承认莱恩身份，到主动使用莱恩身份保护劳动成果。
- 埃文从失败火系学徒，成长为第一代魔法工业技师。
- 巴伦从守旧家臣，变成新工坊契约见证者。
- 灰石领从靠旧设备续命，到开始建立可记录、可培训、可维护的低魔耗工坊体系。

## 情绪承诺

- 前世成果被夺的压抑，在今生保护署名和贡献制度中释放。
- 领民从不信任到第一次围在灰炉边活过寒潮。
- 失败学徒、工匠、领民第一次被记录为创造者，而不是无名劳力。

## 世界承诺

- 这个世界不是简单魔法衰退，而是高魔战争后魔力基础设施崩坏导致的断代社会。
- 第一卷只展示末法现象，不提前解释烬日战争全貌和高魔魔网真相。
- 长篇后续会从灰炉一型逐步进入工坊标准化、金工机械、低压动力、交通物流、城镇基础设施、铭文控制和低魔力网络。

# 03 主角伤口

陆衡前世长期过劳，成果被抢，署名被改，最终意外身亡。

伤口后果：

- 不信权威。
- 不信秘方。
- 对署名和契约敏感。
- 不愿轻易交出核心成果。
- 初期更相信记录和实验，而不是人。
- 面对买断、署名空缺、权威认证时会被触发。

---

# 04 主角欲望

## 外部欲望

让灰石领活过寒潮。

## 中期欲望

建立低魔耗、可复制、可培训、可维护的魔法工业体系。

## 深层欲望

建立一个不会夺走普通劳动者成果的新制度。

## 长篇欲望延展

从第一台灰炉开始，逐步建立适合末法时代的低压、低耗、分布式、可维护、可追责的新魔法工业网络。

---

# 05 世界压力

## 第一卷表层压力

- 寒潮将至。
- 旧魔导暖炉停摆。
- 领地欠税和负债。
- 领民贫困且不信任原身。
- 法师行会垄断魔导器认证。
- 商会想低价吞并或买断技术。
- 劣质魔石输出不稳定。
- 灰石领缺钱、缺粮、缺高阶法师、缺合法维修渠道。

## 底层世界压力

末法不是自然缓慢衰退，而是距今约九十六年前烬日战争导致魔力基础设施崩坏。

高魔时代的魔法工业曾经依赖：

```text
地脉魔力
→ 魔晶矿脉
→ 魔晶炉群
→ 地脉锚点
→ 调律塔
→ 城市魔网
→ 行会认证接入
→ 贵族、工坊、城市公共设备使用
```

烬日战争后，世界没有失去魔法，而是失去了稳定、高纯、高浓度的魔力输入。

```text
过去：高浓度、高纯度、高稳定性的魔力输入。
现在：低浓度、高波动、高杂质的魔力输入。
```

旧设备仍存在，但输入条件失配，边境小领地无法承担高维护成本。

## 六重断层

| 断层 | 内容 | 对剧情的作用 |
|---|---|---|
| 能源断层 | 高魔输入变成低浓度、高波动、高杂质输入 | 解释旧炉失效和灰炉必要性 |
| 材料断层 | 魔石杂质增加，旧合金和铭文载体老化 | 给主角材料学能力切入点 |
| 人才断层 | 地脉调律师、魔晶炉工程师、魔网维护师断代 | 解释行会从研发退化为维护认证 |
| 知识断层 | 高魔知识被炸毁、封存、污染、垄断 | 解释秘方体系和公开记录的价值 |
| 制度断层 | 行会、贵族、商会各自守权 | 形成制度阻力 |
| 思维断层 | 旧世界仍追求更强、更纯、更高阶 | 突出主角低魔耗、低成本、稳定、可维护的工程思维 |

# 06 冲突系统

## 人与环境

寒潮、贫瘠领地、低魔环境、劣质魔石、旧设备损坏。

第一卷具体表现：旧暖炉停摆，领民冻病，灰石领必须在极短时间内找到低成本取暖方案。

## 人与人

巴伦怀疑主角，领民不信任，埃文自卑，工匠不信贵族懂工艺。

第一卷不能让任何人无脑信任主角，所有信任必须由行动、失败承担和阶段性成果换来。

## 人与制度

法师行会认证权、贵族体面、商会买断合同、税务压力。

- 行会代表安全和认证，不是纯反派。
- 贵族体面会妨碍男爵大厅开放给领民。
- 商会合同合法且诱人，但会买断署名和制造权。
- 税务压力逼主角使用莱恩身份。

## 人与历史

高魔时代遗留工业体系在末法时代失效，旧秩序仍不愿承认问题。

真实历史压力来自烬日战争后魔力基础设施崩坏，但第一卷不能提前揭示全貌。

## 人与自我

陆衡不愿承认自己是莱恩，但必须承担莱恩身份的责任。

身份线必须做暗线：

```text
行动上使用莱恩身份，心理上仍保持陆衡视角。
```

# 07 信息释放

## 总原则

第一卷暂不揭示末法历史真相。

第一卷只释放现象和生活层面的解释，不释放烬日战争全貌、地脉锚点崩坏细节、调律塔和城市魔网结构。

## 信息释放顺序

1. 先给现象：暖炉坏、魔石差、设备耗魔高。
2. 再给生活解释：领民冻病、行会维修昂贵、工坊荒废。
3. 再给系统背景：魔导器认证、魔法教育门槛、行会垄断。
4. 后续卷再揭示历史原因：烬日战争、地脉锚点崩坏、城市魔网失配、高魔工业断层。

## 第一卷允许释放

- 旧暖炉停摆。
- 魔石质量下降。
- 行会维修昂贵。
- 旧炉在劣质魔石下不稳定。
- 灰石领撑不过寒潮。
- 低阶火系学徒被旧评价体系淘汰。
- 法师行会认证很重要。
- 商会可以提供现金和渠道，但会要求买断。

## 第一卷禁止释放

- 烬日战争全貌。
- 地脉锚点崩坏细节。
- 调律塔和城市魔网结构。
- 禁忌战略魔导器真相。
- 高魔时代产业体系全貌。
- 低魔力网络的未来蓝图。
- 完整魔法理论。
- 帝国大战和高阶神秘力量。

# 08 人物弧光

## 陆衡 / 莱恩

```text
否认身份
→ 被迫使用莱恩身份
→ 用莱恩身份保护工坊成果
→ 建立新制度
```

第一卷结束时，陆衡不需要彻底接受自己就是莱恩。

他只需要完成价值层面的初步认领：愿意用莱恩·格雷这个名字保护灰石领和工坊参与者劳动成果。

## 巴伦

```text
守旧家臣
→ 怀疑少爷变了
→ 观察主角行动
→ 见证灰炉救人
→ 签署新工坊契约
```

巴伦接受的不是“陆衡”，也不知道魂穿真相，而是接受这个变化后的莱恩至少正在承担格雷家族过去没做到的事。

## 埃文·巴伦

```text
被行会退回的失败学徒
→ 被发现火焰稳定价值
→ 参与实验失败
→ 成为一号炉温控制员
→ 获得署名和岗位身份
```

埃文的爽点不是弱火突然变强，而是弱火被重新定义为工业控温价值。

## 维克多

```text
行会认证法师
→ 质疑灰炉合法性与安全性
→ 被公开测试和炉温记录挑战
→ 无法否认数据，但仍代表行会压力
```

维克多不是小丑反派。他的合理性来自安全、规程和事故责任。

## 赫曼

```text
边境商会代表
→ 看见灰炉商业价值
→ 提出合法且诱人的买断合同
→ 触发主角前世署名伤口
→ 逼出工坊贡献契约
```

赫曼不是低级奸商。他的合同必须能真正解决灰石领短期现金压力，同时危险地抹除参与者署名和制造权。

# 09 章节设计

第一卷前 30 章分三段：

1. 第 1-10 章：不承认身份，信息拼图，第一次失败。
2. 第 11-20 章：技术试错，人际怀疑，被迫承担。
3. 第 21-30 章：灰炉兑现，行会质疑，商会买断，初步接受身份。

每章必须回答四问：

- 主角现在知道什么？
- 主角为什么行动？
- 世界信息如何出现？
- 心理、关系或局面是否变化？

章节五段必须完整：

```text
章节目标
→ 阻碍
→ 行动
→ 变化
→ 钩子
```

# 10 场景设计

重要场景必须是冲突单元，不是说明书。

核心场景方向：

- 巴伦试探主角身份。
- 主角拆旧魔导炉。
- 第一批耐火砖炸裂。
- 埃文第一次控温失败。
- 税务官逼迫签字。
- 寒潮夜灰炉救人。
- 行会公开测试。
- 商会买断合同。
- 工坊贡献契约诞生。

世界信息必须通过场景压力出现，例如账本、旧炉封印、魔石测试、失败实验、领民冻病、维克多质疑和赫曼合同。

# 11 爽点设计

## 爽点循环

```text
旧体系判断某人/某物无价值
→ 主角用工程方法重新定义价值
→ 经过失败和记录
→ 小规模成功
→ 旧体系质疑
→ 公开结果打脸
→ 新制度留下成果
```

## 第一卷重点爽点

- 弱火学徒变控温员。
- 劣质魔石变低魔耗燃料。
- 泥砖变耐火砖。
- 灰炉比贵族旧炉更稳定。
- 主角拒绝买断并建立署名制度。
- 旧评价体系看不上的低阶参与者第一次被记录在工坊成果里。

## 禁止爽点误用

- 不要让埃文弱火突然变强火。
- 不要让灰炉一次完美成功。
- 不要让主角靠口号赢商会。
- 不要让维克多为了被打脸而降智。
- 不要让配角立刻崇拜主角。

# 12 改稿审计

## 当前高风险点

- 主角接受身份过快。
- 配角接受主角过快。
- 技术进展过快。
- 第一卷过早扩展宏大世界观。
- 法师行会脸谱化。
- 商会买断过于低级。
- 过早进入金系、风系交通、水系城建或低魔力网络。
- 把烬日战争、地脉锚点、调律塔、城市魔网讲成第一卷说明书。
- 把魔法能力工业化写成元素设定罗列，而不是岗位、工序、失败和成本。

## 修订原则

- 外部事件可以快，心理接受必须慢。
- 主角可以被迫行动，但不能马上认同新身份。
- 配角必须通过行动结果逐步改变态度。
- 第一卷只聚焦灰炉、寒潮、身份承认、贡献契约。
- 世界观只通过人物需要和场景压力释放。
- 技术进展必须经历观察、建模、测试、失败、记录、改进。
- 旧秩序必须有合理性，不能靠降智给主角垫脚。
- 长篇工业路线可以作为底层约束，但第一卷不能提前展开。

# 13 后续工业路线约束

长篇工业路线如下：

```text
生存热工
→ 工坊标准化
→ 金工与机械基础
→ 低压动力
→ 交通物流
→ 城镇基础设施
→ 铭文仪表与控制
→ 低魔力网络
```

第一卷只允许正式进入第一阶段：生存热工。

其余阶段可通过工具损坏、旧零件不足、通风问题、冷却问题、认证问题轻微埋伏笔，但不能抢主线。

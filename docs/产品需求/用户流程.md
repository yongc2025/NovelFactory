## 1. 仪表盘（Dashboard）

### 1.1 页面功能

| 功能 | 描述 |
|------|------|
| 项目列表 | 展示所有项目，支持搜索和筛选 |
| 统计数据 | 总项目数、生成中、已完成、总字数 |
| 快速操作 | 新建项目、进入项目、删除项目 |
| 状态展示 | 项目当前阶段、进度百分比 |

### 1.2 用户操作

```
用户打开系统
    │
    ▼
┌─────────────────────────────────────┐

│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐  │

│  └─────┘ └─────┘ └─────┘ └─────┘  │
├─────────────────────────────────────┤
│  📋 项目列表                        │

│  │ 项目1 | 状态 | 进度 | 操作   │  │
│  │ 项目2 | 状态 | 进度 | 操作   │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

| 模式 | 步骤数 | 适用场景 |
|------|--------|----------|

| 完整模式 | 8 步 | 精细控制，自定义参数 |

### 2.4 参数说明

| 级别 | 参数 | 必填 | 说明 |
|------|------|------|------|
| L1 | 灵感/前提 | ✅ | 50-200字核心冲突 |
| L1 | 目标平台 | ✅ | 番茄/小红书/短剧 |
| L1 | 篇幅类型 | ✅ | short/medium/long/comic |
| L2 | 题材定位 | ❌ | 大类+细分 |
| L2 | 目标读者 | ❌ | female/male/general |
| L2 | 内容基调 | ❌ | 爽文/虐文/甜文 |
| L3 | 角色预设 | ❌ | 主角/反派/配角 |
| L4 | 世界观约束 | ❌ | 时空背景/参考作品 |
| L5 | 生成策略 | ❌ | 字数/章数/爽点密度 |

## 4. 流水线控制

### 4.1 启动流水线

```
用户点击 [启动流水线]
    │
    ▼
┌─────────────────┐
│  确认对话框     │
│  "确定启动？"   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  创建后台任务   │
│  进入生成状态   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  轮询进度       │
│  更新进度条     │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  阶段完成       │
│  等待用户确认   │
└────────┬────────┘
         │
         ▼
    [用户确认/修订]
```

### 4.2 阶段确认流程

```
阶段生成完成
    │
    ▼
┌─────────────────┐
│  展示结果       │
│  (选题/大纲/...)│
└────────┬────────┘
         │
         ├─→ [确认通过] → 进入下一阶段
         │
         ├─→ [提出反馈] → AI 重新生成
         │
         └─→ [手动编辑] → 保存修改
```

### 4.3 支持的操作

| 操作 | 描述 | 影响范围 |
|------|------|----------|
| 确认 | 接受当前结果，进入下一阶段 | 当前阶段 |
| 重新生成 | AI 重新生成当前阶段 | 当前阶段 |
| 提出反馈 | 输入修改意见，AI 重新生成 | 当前阶段 |
| 手动编辑 | 直接编辑结果 | 当前阶段 |
| 回退 | 返回上一阶段重新生成 | 当前+后续阶段 |

## 6. 导出发布

### 6.1 导出流程

```
用户点击 [导出]
    │
    ▼
┌─────────────────┐
│  选择格式       │
│  - Markdown     │
│  - TXT (番茄)   │
│  - EPUB         │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  选择范围       │
│  - 全书         │
│  - 指定章节     │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  下载文件       │
└─────────────────┘
```

### 6.2 格式说明

| 格式 | 平台 | 特点 |
|------|------|------|
| Markdown | 通用 | 包含元数据、目录、正文 |
| TXT | 番茄小说 | 纯文本，无格式 |
| EPUB | 电子书 | 标准电子书格式 |

## 7. 异常处理

### 7.1 生成失败

```
生成失败
    │
    ▼
┌─────────────────┐
│  显示错误信息   │
│  "第X章生成失败"│
└────────┬────────┘

         └─→ [取消] → 取消整个任务
```

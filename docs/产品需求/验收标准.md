# NovelFactory 验收标准

> 版本：1.0 | 更新日期：2026-06-12
> 本文档定义系统交付的验收标准，是最终验收的依据。

---

## 验收原则

1. **以本文档为准**：所有验收必须基于本文档定义的标准
2. **逐项验证**：每个验收项必须独立验证
3. **证据留存**：验收过程需截图或录屏留存
4. **问题记录**：未通过项需记录问题详情

## 验收流程

### 验收步骤

1. **准备阶段**
   - 确认环境配置正确
   - 准备测试数据
   - 确认验收人员

2. **执行阶段**
   - 按验收项逐项验证
   - 记录验证结果
   - 截图/录屏留存

3. **问题处理**
   - 记录未通过项
   - 分析问题原因
   - 提交修复任务

4. **验收确认**
   - 所有验收项通过
   - 签署验收报告
   - 交付使用

## 3. 女主/感情线模式（可选 — 男频适用）

| 标签 | 说明 | 适用题材 | 读者接受度 |
|------|------|---------|-----------|
| 无女主 | 纯事业线，无感情戏或感情戏极少 | 升级流、争霸、探险 | ⭐⭐⭐⭐⭐ 最安全，无争议 |
| 单女主 | 一对一专一感情线，女主戏份重要 | 都市、仙侠、玄幻 | ⭐⭐⭐⭐⭐ 主流选择，接受度最高 |
| 后宫 | 多女主，每位女主有独立人设和故事线 | 玄幻、都市、架空历史 | ⭐⭐⭐ 有固定受众，但需写好差异化 |
| 暧昧 | 有感情戏但不确定关系，若即若离 | 都市、校园、轻小说 | ⭐⭐⭐ 轻松向，但容易烂尾 |
| 红颜知己 | 不谈恋爱，但有深度情感羁绊 | 武侠、仙侠、权谋 | ⭐⭐⭐⭐ 文艺向，适合高口碑作品 |
| 先婚后爱 | 先结婚/契约关系，后产生真感情 | 都市、古言、豪门 | ⭐⭐⭐⭐ 女频经典模式，男频偶用 |

**平台差异提示：**
- 起点中文网：单女主和无女主最安全，后宫需谨慎
- 番茄小说：后宫接受度相对较高，但单女主仍是主流
- 晋江文学城：严格要求一对一，后宫文基本没有市场

## 4. 女主性格模板（有女主/后宫时使用）

| 标签 | 一句话描述 | 适用场景 | 备注 |
|------|-----------|---------|------|
| 冰山美人 | 外冷内热，只对男主破功 | 仙侠、玄幻、都市 | 经典万能型，搭配逆袭男主效果最佳 |
| 天真活泼 | 开朗乐观，是男主的开心果 | 校园、轻松向、甜宠 | 适合轻松日常，不适合沉重剧情 |
| 温柔贤惠 | 默默守护，无条件支持男主 | 都市、年代、种田 | 后宫文中常做"正宫"角色 |
| 女强人 | 独立自主，与男主势均力敌 | 都市、商战、星际 | 现代感强，适合双强设定 |
| 腹黑绿茶 | 表面柔弱实则心机深，亦正亦邪 | 宫斗、宅斗、权谋 | 争议型角色，写好了是经典，写砸了被骂 |
| 青梅竹马 | 从小一起长大，感情基础深厚 | 校园、都市、仙侠 | 情感铺垫自然，读者代入感强 |
| 傲娇大小姐 | 嘴上嫌弃实际在意，反差萌 | 都市、校园、轻喜剧 | 日系ACG经典属性，国内也很受欢迎 |
| 呆萌吃货 | 天然呆+吃货属性，无攻击性 | 轻松向、日常、甜宠 | 减压型角色，适合轻松氛围 |
| 御姐型 | 成熟性感，比男主年长或心理成熟 | 都市、职场、现代 | 适合有一定阅历的男主 |
| 侠女型 | 行侠仗义，英姿飒爽，巾帼不让须眉 | 武侠、仙侠、江湖 | 双侠侣设定，适合武侠仙侠 |

**后宫差异化建议（如选择后宫模式）：**
- 至少保证 3-4 位女主性格完全不同
- 每位女主应有独立的故事线和成长弧
- 避免"花瓶化"，每位女主都需要有独特价值
- 感情发展节奏要有差异，不要同时推进

## 5. 反派设计模式（系统自动生成参考）

### 反派模式分类

| 模式 | 说明 | 最佳适用题材 | 推荐度 |
|------|------|-------------|--------|
| 单元反派 | 每卷/每个 arc 一个独立反派，打完换下一个 | 升级流、探险、玄幻 | ⭐⭐⭐⭐⭐ 节奏紧凑，适合长篇 |
| 贯穿反派 | 从头到尾的大 boss，最终决战 | 仙侠、权谋、热血 | ⭐⭐⭐⭐ 史诗感强，但需控制节奏 |
| 反派联盟 | 多个势力交织，敌友难辨 | 权谋争霸、末世、架空历史 | ⭐⭐⭐⭐ 格局大，适合高智商剧情 |
| 亦正亦邪 | 灰色地带角色，有苦衷的反派 | 都市、武侠、仙侠 | ⭐⭐⭐⭐ 深度感强，提升作品口碑 |
| 宿命之敌 | 旗鼓相当的对手，互相成就 | 热血战斗、竞技、武侠 | ⭐⭐⭐⭐⭐ 张力最强，经典模式 |
| 家族/宗门势力 | 以家族或宗门为单位的敌对势力 | 玄幻、仙侠、都市 | ⭐⭐⭐⭐ 适合升级流层层递进 |
| 黑化队友 | 原本的同伴/挚友反目成仇 | 热血、仙侠、校园 | ⭐⭐⭐⭐ 情感冲击力强 |
| 幕后黑手 | 前期不露面，后期揭示真面目 | 悬疑、权谋、科幻 | ⭐⭐⭐⭐ 反转感强，适合烧脑剧情 |

### 按题材推荐反派设计模式

| 题材 | 推荐组合 | 说明 |
|------|---------|------|
| 玄幻升级流 | 单元反派 + 贯穿反派 | 每个境界一个对手，最终大boss贯穿 |
| 都市逆袭 | 单元反派 + 家族势力 | 每个阶段面对不同阶层的打压 |
| 仙侠 | 宿命之敌 + 亦正亦邪 | 修仙路上的对手和道心之争 |

| 悬疑推理 | 幕后黑手 + 单元反派 | 每个案件独立，背后有大阴谋 |

## 6. 各平台风格差异

### 番茄小说

| 维度 | 说明 |
|------|------|
| **核心特点** | 免费阅读、算法推荐为主、节奏极快、下沉市场 |
| **热门题材** | 都市逆袭、战神归来、赘婿打脸、重生复仇、甜宠、穿越 |
| **节奏要求** | 前3章必须有爽点/冲突，每章结尾留钩子，不能"慢热" |
| **字数要求** | 单章 2000-3000 字为佳，总字数 100-300 万字，长尾效应 |
| **读者画像** | 下沉市场为主，年龄偏大（25-45），碎片化阅读，追求即时爽感 |
| **禁忌** | 开篇慢热、大段心理描写、文青病、说教、政治敏感 |
| **收益模式** | 广告分成，靠阅读量，需高更新频率（日更 4000+ 字） |

### 起点中文网

| 维度 | 说明 |
|------|------|
| **核心特点** | 付费阅读、精品化、体系完整、核心男频阵地 |
| **热门题材** | 玄幻修仙、科幻、历史架空、游戏、都市异能、武侠 |
| **节奏要求** | 允许铺垫但不能太久（前 3 万字见真章），世界观构建重要 |
| **字数要求** | 单章 3000-5000 字，总字数 200-500 万字，精品长篇 |
| **读者画像** | 核心网文读者（18-35），口味挑剔，愿意付费，追求深度 |
| **禁忌** | 逻辑硬伤、抄袭融梗、烂尾、太监（断更）、小白文过于降智 |
| **收益模式** | 订阅分成 + 打赏，需稳定更新和口碑积累 |

### 晋江文学城

| 维度 | 说明 |
|------|------|
| **核心特点** | 女频为主、耽美/百合、精品化、社区氛围强 |
| **热门题材** | 言情（古言/现言）、耽美、百合、无 CP、穿越重生、悬疑 |
| **节奏要求** | 注重文笔和情感铺垫，允许慢热但需有吸引力 |
| **字数要求** | 单章 3000-6000 字，总字数 30-150 万字（比男频短） |
| **读者画像** | 女性为主（18-30），高学历偏多，注重文笔和三观，社区感强 |
| **禁忌** | 三观不正（争议大）、抄袭、融梗、人设崩塌、烂尾、过度色情 |
| **收益模式** | VIP 章节订阅，靠收藏和口碑，完结后仍有长尾收入 |

### 小红书（网文/小说板块）

| 维度 | 说明 |
|------|------|
| **核心特点** | 短内容引流、种草式推荐、视觉化、社交传播 |
| **热门题材** | 短篇甜文、虐文、穿书、重生、ABO、同人、追妻火葬场 |
| **节奏要求** | 极度压缩，开头即高能，500 字内抓住读者 |
| **字数要求** | 短篇 1-5 万字为主，中篇 10-30 万字，适合引流到其他平台 |
| **读者画像** | 年轻女性（16-28），追星/二次元群体，追求情绪价值 |
| **禁忌** | 节奏拖沓、文字不够精炼、封面/标题不吸引人 |
| **收益模式** | 引流为主，通过小红书种草后导流到番茄/晋江等平台阅读 |

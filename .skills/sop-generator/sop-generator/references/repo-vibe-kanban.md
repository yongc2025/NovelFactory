# 仓库摘要：BloopAI/vibe-kanban

## 项目定位（README 摘要）

- 项目是一个“live GPT task board”，强调可用自然语言管理与执行任务  
- 支持多模型供应商（OpenAI / Anthropic）  
- 以本地优先为导向（local-first）  

## 快速运行要点（README 摘要）

- 安装依赖：`pnpm install`  
- 启动开发：`pnpm dev`  
- 需要配置模型供应商的 API Key
